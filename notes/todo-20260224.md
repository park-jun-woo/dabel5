# DABEL5 보완 과제 목록

NASA Phase A 수준으로 끌어올리기 위한 보완 항목. 우선순위순.

---

## 1. 더블링 주기 상세 도출 [최우선]

시리즈 전체의 기둥. "2년"이라는 숫자의 공정별 근거가 없음.

- [ ] 율속 단계(rate-limiting step) 식별
  - 거울 기판 압연 속도 (m²/hr)
  - Al 코팅 증착 속도 (m²/hr)
  - 터빈 블레이드 주조/가공 시간
  - 구조재 용접/조립 처리량
  - 제련 처리량 (톤/일) vs 거울 1기 소요 질량
- [ ] 공정별 병목 분석 → 더블링 주기 재계산
- [ ] 민감도: 병목 공정이 2배 느릴 때 더블링 주기 변화

## 2. 1986 DA 조성 민감도 분석

레이더 알베도 기반 90%+ Fe-Ni는 추정치. 확정 아님.

- [ ] 시나리오별 타임라인: Fe-Ni 90% / 70% / 50% / 30%
- [ ] 각 시나리오에서 K1.0 도달 연수 재계산
- [ ] 30% 이하일 때 대안 소행성 후보 리스트 (16 Psyche, 216 Kleopatra 등)
- [ ] Psyche 미션 데이터 반영 계획 (2029 도착 예정)

## 3. 첫 모듈 질량 예산 (Mass Budget)

모듈 1기의 상세 질량 분해가 없음.

- [ ] 서브시스템별 질량
  - 집광 거울 (면적, 두께, 소재, 코팅)
  - 터빈 + 발전기
  - 방열판
  - 제련로/가공 설비
  - 구조 프레임
  - 제어/통신/AI 컴퓨트
  - 거주 모듈 (해당 시)
- [ ] 총 질량 → 1986 DA 자원으로 몇 기 제작 가능한지 역산
- [ ] 질량 대비 출력 (W/kg) 산출

## 4. EML5 시연 미션 설계

Bootstrap 첫 단계의 구체적 미션 아키텍처.

- [ ] 미션 목표: 거울 1장 배치 + 소규모 태양열 제련 실증
- [ ] 발사 질량 / 발사체 (Starship 기준)
- [ ] LEO → EML5 전이 궤도 설계 (delta-V, 이동 시간)
- [ ] 초기 자원: 달 레골리스 vs 지구 발사 프리팹
- [ ] 미션 비용 개략 추정 (Order of magnitude)
- [ ] 성공 기준 정의

## 5. 실패 모드 분석 (FMEA)

단일 실패점(single point of failure) 식별.

- [ ] 제련소 고장 시 전체 루프 중단 여부
- [ ] 터빈 고장 시 대체 전력
- [ ] 수송선 손실 시 원료 공급 중단 기간
- [ ] AI 자율 제어 실패 시 폴백 모드
- [ ] 각 실패 모드의 심각도 / 발생 확률 / 대응 방안

## 6. TRL 로드맵

개별 기술의 현재 TRL과 필요 TRL 간 갭 분석.

- [ ] 우주 SMR (50-100 kW): NASA FSP → 비행 실증 경로
- [ ] NTP 추진: DARPA DRACO → 운용급 타임라인
- [ ] 미소중력 대규모 채굴: OSIRIS-REx 수백 g → 수만 톤 스케일 갭
- [ ] 진공 태양열 제련: 지상 실험 → 우주 실증 경로
- [ ] 자기복제 제조: 개념 → 지상 프로토타입 → 우주 실증
- [ ] 각 기술별 2038년까지의 마일스톤

## 7. 방사선 환경 및 장비 수명

SEL5에서의 우주 환경이 장기 운용에 미치는 영향.

- [ ] GCR/SEP 플럭스 at 1 AU (Van Allen 벨트 외부)
- [ ] 거울 반사율 열화 모델 (CME 스퍼터링, 미소 운석)
- [ ] 터빈/베어링/전자장비의 방사선 내성
- [ ] 예상 교체 주기 → 유지보수 물류 설계
- [ ] 거울 코팅 재증착 현장 수리 가능성

## 8. 열 사이클 피로

궤도/운용 조건에 따른 반복 온도 변화의 구조적 영향.

- [ ] 거울: 그림자 진입/이탈 시 온도 변동 범위
- [ ] 터빈: 시동/정지 사이클 빈도와 블레이드 수명
- [ ] 배관/열교환기: 열 팽창 계수 차이에 의한 피로
- [ ] Fe-Ni 합금의 우주 환경 피로 데이터 조사

## 9. 미소 파편 및 충돌 리스크

수십만 기 모듈 운용 시의 우주 쓰레기 문제.

- [ ] SEL5 영역의 자연 미소 운석 플럭스
- [ ] 모듈 간 충돌 확률 (50-100 km 간격 기준)
- [ ] 거울 면의 미소 운석 관통 허용 한계
- [ ] 능동 파편 회피 시스템 필요 여부
