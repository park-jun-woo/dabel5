## 3. Leg 1: 1986 DA → EML5/SEL5 (원광 수송)

### 3.1. 1986 DA 궤도 파라미터

| 항목 | 값 |
|------|-----|
| 분류 | M형 Amor NEA |
| 반장축 (a) | 2.81 AU |
| 이심률 (e) | 0.58 |
| 근일점 (q) | 1.17 AU |
| 원일점 (Q) | 4.46 AU |
| 궤도 경사 (i) | 4.3° |
| 궤도 주기 (P) | 4.71년 (1,720일) |
| 2038년 최근접 | **0.21 AU (~3,150만 km)** |

### 3.2. 전이 궤도

**호만 전이 (근일점 → 지구 궤도)**

근일점(1.17 AU)에서 출발, 1 AU 도착:

```
전이 타원: a_t = (1.17 + 1.0) / 2 = 1.085 AU

근일점(1.17 AU)에서 속도:
  v_ast = √(GM/a) × √((1-e)/(1+e)) ← 소행성 궤도
  v_ast = 29.78/√2.81 × √(0.42/1.58) = 17.76 × 0.516 = 9.16 km/s ... (*)

  (*) 실제로는 vis-viva로 계산:
  v_ast at 1.17 AU = √(GM × (2/1.17 - 1/2.81))
                   = 29.78 × √(1.709 - 0.356)
                   = 29.78 × √1.353
                   = 29.78 × 1.163
                   = 34.63 km/s ... (이건 r=1.17에서의 속도)

  잠깐, 단위를 맞추자.
  GM_sun = 1.327 × 10²⁰ m³/s², 1 AU = 1.496 × 10¹¹ m

  v = √(GM × (2/r - 1/a))
  r = 1.17 AU, a = 2.81 AU
  v = √(1.327e20 × (2/(1.17×1.496e11) - 1/(2.81×1.496e11)))
    = √(1.327e20 × (1.143e-11 - 2.378e-12))
    = √(1.327e20 × 9.05e-12)
    = √(1.201e9)
    = 34,660 m/s = 34.66 km/s

전이 궤도 at r=1.17 AU (a_t=1.085 AU):
  v_t = √(GM × (2/r - 1/a_t))
      = √(1.327e20 × (1.143e-11 - 6.161e-12))
      = √(1.327e20 × 5.27e-12)
      = √(6.99e8)
      = 26,440 m/s = 26.44 km/s

Δv₁ = |34.66 - 26.44| = 8.22 km/s ... (소행성 출발)
```

이 값은 소행성 궤도에서 직접 전이 시의 Δv다. 실제로는 LEO에서 출발하므로 지구 탈출 Δv가 포함되어 ~7.1 km/s (LEO 기준)이 된다.

### 3.3. Δv 및 추진 옵션

| 방식 | Isp | Δv 7.1 km/s 시 질량비 | 특징 |
|------|-----|---------------------|------|
| 화학 (LH2/LOX) | 450 s | e^(7100/4414) = **5.0** | 추진제가 화물의 4배 → ❌ |
| **NTP** | **900 s** | e^(7100/8829) = **2.24** | 추진제 = 화물의 1.24배 → ✅ |
| **NEP** | **3,000 s** | e^(7100/29430) = **1.27** | 추진제 = 화물의 0.27배 → ✅✅ |

**질량비 = (초기질량/최종질량). NEP는 20만 톤 화물에 추진제 ~5.4만 톤.**

### 3.4. EML5 vs SEL5 도착 비교

| | → EML5 | → SEL5 |
|---|---|---|
| 도착 타겟 크기 | ~수만 km (제한적) | **수백만 km (거대)** |
| 지구 중력 포획 필요 | **예** (힐 영역 진입) | **아니오** (태양 궤도에 머무름) |
| 추가 Δv | 지구 접근 브레이킹 ~0.5-1 km/s | 거의 없음 (넓은 타겟) |
| 통신 지연 | 1.3초 (지구 관제) | 8분 20초 (AI 자율) |
| 수령 인프라 | Bootstrap 규모 | **대규모 스웜** |

**결론:** Bootstrap 단계에서는 EML5, Scale-up 이후에는 SEL5가 최적. SEL5는 타겟이 넓어 궤도 조준 부담이 적음.

## 4. Leg 2a: EML5 → SEL1 (Bootstrap 단계)

### 4.1. 기하학

```
          ☀️
          │
       [SEL1] ← 목적지
          │ ~150만 km
          │
        🌍 ─── ~38만 km ─── [EML5] ← 출발
```

- 총 이동 거리: ~150만~190만 km (기하학적 배치에 따라)
- 두 점 모두 지구 힐 영역(~150만 km) 내/경계에 위치
- **행성간 전이가 아닌 지구 근방 전이**

### 4.2. 전이 방식

**저에너지 전이 (Low-Energy Transfer)**

지구-달-태양 3체 문제의 동역학 구조(invariant manifold)를 활용한 저에너지 경로:

- EML5 → 지구-달 시스템 탈출 → SEL1 해일로 궤도 진입
- Δv: **~0.3~0.8 km/s** (전이 궤도 설계에 따라)
- 전이 시간: **2~6개월**
- 참고: SOHO, DSCOVR 등 실제 SEL1 미션이 이 접근법 사용

**솔라 세일 전이**

기후 패널을 솔라 세일로 운용:
- EML5에서 지구 중력권 탈출 → 태양 복사압으로 SEL1 접근
- 패널 면질량 40 g/m² → 특성 가속도 0.23 mm/s² (매우 우수)
- Δv 축적 속도: **51일에 1 km/s**
- 전이 시간: **1~3개월**

### 4.3. 판정

**매우 타당.** 거리가 짧고(~200만 km), 저에너지 전이 경로가 잘 연구되어 있으며, 실제 SEL1 미션 사례가 다수. Bootstrap 단계의 소규모 시험용 차폐막 배치에 최적.

## 5. Leg 2b: SEL5 → SEL1 (Scale-up 단계) ⭐ 핵심 구간

### 5.1. 문제 정의

- 출발: SEL5 (1 AU, 지구 뒤 60°)
- 도착: SEL1 (1 AU, 지구 앞 ~0.01 AU)
- **태양 궤도상 위상차: ~60°**
- **호 길이: ~1.57억 km (1 AU × π/3)**

이건 "바로 옆으로 옮기기"가 아니라 **지구-태양 거리에 맞먹는 행성간 수송**이다.

### 5.2. 방법 1: 통상 추진 (phasing orbit)

궤도 반장축을 줄여 공전 주기를 단축, 위상을 따라잡는 방식.

```
목표: 60° 위상 전진 (SEL5 → 지구 근방)

접근: 안쪽 타원궤도로 전이 → 빠른 공전 → 위상 따라잡기 → 원궤도 복귀

전이 궤도 설계 (2년 전이):
  원궤도: a = 1 AU, P = 365.25일
  전이궤도: P_t = P × (1 - 60°/360°) × (n_orbits / n_orbits)

  1회전 전이: 완료 시 300°만 진행 = 360° - 60°
    P_t = 365.25 × (300/360) = 304.4일
    a_t = (P_t/P)^(2/3) = (304.4/365.25)^(2/3) = 0.833^(2/3) = 0.886 AU

  vis-viva at r=1 AU (a_t=0.886 AU):
    v_t = √(GM × (2/r - 1/a_t))
        = 29.78 × √(2 - 1/0.886)
        = 29.78 × √(2 - 1.129)
        = 29.78 × √0.871
        = 29.78 × 0.933
        = 27.79 km/s

  Δv₁ = |29.78 - 27.79| = 1.99 km/s (궤도 진입)
  Δv₂ = 1.99 km/s (원궤도 복귀)
  총 Δv = ~3.98 km/s
```

1회전 전이(~304일)에 Δv ~4 km/s. **수천만 톤 벌크 화물에 비현실적.**

```
느린 전이 (다수 회전):
  5회전 전이: 5P_t = 5P - (60°/360°)×P
    P_t = P × (5 - 1/6) / 5 = 365.25 × 29/30 = 353.1일
    a_t = (353.1/365.25)^(2/3) = 0.9668^(2/3) = 0.978 AU

  v_t at 1 AU = 29.78 × √(2 - 1/0.978) = 29.78 × √(2 - 1.022) = 29.78 × 0.988 = 29.44 km/s
  Δv₁ = |29.78 - 29.44| = 0.34 km/s
  총 Δv = ~0.68 km/s
  전이 시간: ~4.8년
```

**5회전 전이: Δv ~0.68 km/s, 4.8년.** Δv는 낮지만 시간이 너무 길다.

### 5.3. 방법 2: 솔라 세일 (자력 항해) ← 최적해

기후 패널의 솔라 세일 성능:

| 항목 | 값 |
|------|-----|
| 패널 두께 | 5 μm Fe-Ni |
| 면질량 (σ) | 0.04 kg/m² (40 g/m²) |
| 면적/질량비 | **25 m²/kg** |
| 태양 복사압 (1 AU) | 4.56 × 10⁻⁶ N/m² |
| 반사 시 힘 | 2 × 4.56 × 10⁻⁶ = 9.12 μN/m² |
| **특성 가속도** | **a_c = 9.12e-6 / 0.04 = 2.28 × 10⁻⁴ m/s² ≈ 0.23 mm/s²** |

비교:

| 솔라 세일 | 특성 가속도 | DABEL5 패널 대비 |
|----------|-----------|----------------|
| IKAROS (JAXA, 2010) | 0.001 mm/s² | **1/230** |
| LightSail 2 (TPS, 2019) | 0.058 mm/s² | **1/4** |
| NEA Scout (NASA, 2022) | 0.011 mm/s² | **1/21** |
| **DABEL5 기후 패널** | **0.23 mm/s²** | **기준** |
| 이론적 고성능 세일 | 0.1~1 mm/s² | 0.4~4.3× |

**DABEL5 패널은 기존 실증 세일 대비 4~230배 빠르다.** 40 g/m²의 극박 Fe-Ni 필름이 기후 차폐막인 동시에 고성능 솔라 세일이다.

### 5.4. 솔라 세일 전이 분석

**Δv 축적 속도:**

```
Δv = a_c × t
1 km/s 축적: t = 1000 / 0.000228 = 4.39 × 10⁶ s ≈ 51일
```

**60° 위상 이동 전략:**

솔라 세일은 추력 벡터를 조절해 궤도를 변경한다. 태양 방향에 대한 패널 각도를 제어하면:
- 궤도 안쪽으로 나선 → 공전 주기 단축 → 위상 전진
- 도착 위치에서 다시 바깥으로 나선 → 1 AU 원궤도 복귀

```
나선 전이 시나리오:
  Phase 1: 안쪽 나선 (1 AU → ~0.95 AU), ~60일
  Phase 2: 위상 드리프트 (~0.95 AU에서 공전), ~120~180일
    위상 드리프트율: ~7°/월 (주기차 ~27일/년)
    60° 드리프트: ~8.5개월
  Phase 3: 바깥 나선 (0.95 AU → 1 AU), ~60일
  Phase 4: SEL1 해일로 궤도 진입, ~30일

  총 전이 시간: ~10~12개월
  추진제: 제로
```

### 5.5. 대규모 운용

8,000만 톤의 패널을 SEL1에 배치해야 한다. 개별 패널이 자력 항해하므로:

| 시나리오 | 연간 생산량 | 연간 배치량 | SEL1 완성 시간 |
|---------|-----------|-----------|--------------|
| 1,000 모듈 가동 | ~수백만 톤/년 | ~수백만 톤/년 | 수십 년 |
| 10,000 모듈 가동 | ~수천만 톤/년 | ~수천만 톤/년 | **수 년** |
| 200,000 모듈 가동 | ~수억 톤/년 | 병목 없음 | **1년 미만** |

패널은 연속적으로 생산·출발시킬 수 있다. 전이 중인 패널이 수백만 장 동시에 태양계를 항해하는 그림. **병렬 수송이므로 전이 시간(12개월)은 초기 지연일 뿐, 처리량 병목이 아니다.**

### 5.6. 판정

**타당.** 기후 패널의 극저 면질량(40 g/m²)이 기존 솔라 세일 대비 압도적 성능을 제공하여, 추진제 없이 6~12개월 내 SEL5 → SEL1 전이 가능. 대규모 운용 시 병렬 수송으로 처리량 제한 없음.

## 6. SEL1 궤도 유지 (Station-keeping)

### 6.1. SEL1 불안정성

SEL1은 불안정 라그랑주점이다. 섭동이 있으면 궤도에서 이탈한다.

| 항목 | 값 |
|------|-----|
| 유형 | 불안정 (saddle-type) |
| 안정 시간 (무보정) | ~23일 (e-folding time) |
| 해일로 궤도 반경 | ~25만~80만 km |
| 스테이션키핑 Δv (종래) | ~2~5 m/s/년 (화학 추진) |

### 6.2. 솔라 세일 궤도 유지

기후 패널은 항상 태양을 향한다 (차폐가 목적). 따라서 태양 복사압이 지속적으로 작용.

```
복사압 가속도: 0.23 mm/s² = 0.23 × 10⁻³ m/s²
연간 Δv 예산: 0.23e-3 × 86400 × 365 = ~7,200 m/s/년 (이론상 최대)

필요 스테이션키핑 Δv: ~2~5 m/s/년
가용 Δv: ~7,200 m/s/년

여유도: ~1,000배 이상
```

패널 각도를 0.1° 미만만 조절해도 충분한 스테이션키핑 Δv를 생성. **추진제 완전 불필요.**

### 6.3. 자세 제어

| 제어 수단 | 방식 |
|----------|------|
| 1차 | 패널 각도 조절 (복사압 벡터 변경) |
| 2차 | 패널 가장자리 반사율 차등 코팅 (토크 생성) |
| 3차 | 인접 패널 간 전자기 커플링 (옵션) |

### 6.4. 판정

### 7.2. 지구 그림자

```
지구 본영 직경 at 384,400 km:
  r = 6,371 - 384,400 × 689,629 / 149,600,000
  = 6,371 - 1,772 = 4,599 km (반경)
  → 직경 ~9,200 km

진입 조건: 황도 위도 < arctan(4,599 / 384,400) ≈ 0.69°
달 궤도 경사 5.14° → 승교점/하강점 근처에서만 발생
```

| 항목 | 값 |
|------|-----|
| 빈도 | 연 2~3회 |
| 1회 최대 지속 | ~2.5시간 (본영) / ~4.3시간 (반영 포함) |
| 연간 총 다운타임 | 3~12시간 |
| **연간 가동률** | **99.86~99.97%** |

## 10. 핵심 통찰: 기후 패널 = 솔라 세일

이 설계에서 가장 우아한 점: **기후 차폐막과 수송 수단이 동일한 물체.**

| 기능 | 물리 원리 | 동일 여부 |
|------|----------|----------|
| 기후 차폐 | 태양광 반사/차단 | ☀️ → 패널 |
| 솔라 세일 추진 | 태양 복사압 | ☀️ → 패널 |
| SEL1 궤도 유지 | 태양 복사압 제어 | ☀️ → 패널 |

세 기능이 전부 **"얇은 막이 태양빛을 받는다"**는 동일한 물리에 기반한다. 별도 추진 시스템, 별도 궤도 유지 시스템이 불필요. 막 자체가 화물이자 엔진이자 스테이션키퍼.

## 12. 결론

DABEL5 공급망의 궤도역학은 전 구간에서 물리적으로 타당하다.

| 구간 | 판정 | 핵심 근거 |
|------|------|----------|
| 1986 DA → EML5/SEL5 | **타당** | 표준 행성간 전이, NTP/NEP 기술 개발 중 |
| EML5 → SEL1 | **매우 타당** | 200만 km, 저에너지 전이 실증 사례 다수 |
| SEL5 → SEL1 | **타당** | 패널 = 솔라 세일 (0.23 mm/s²), 추진제 제로 |
| SEL1 궤도 유지 | **매우 타당** | 복사압 여유도 1,000배 이상 |

가장 도전적인 구간(SEL5 → SEL1, 1.5억 km)을 해결하는 열쇠는 패널 자체의 물성에 있다. 40 g/m²의 극박 Fe-Ni 필름은 기존 실증 솔라 세일 대비 4~230배의 특성 가속도를 가지며, 추진제 없이 6~12개월 내 전이를 완료한다.

> **"패널이 화물이자 엔진이다."**
