# 스윙스테이션 (Swing Station)

> **반대 방향으로 도는 쌍 로터가 원심력으로 기계적 스윙바이를 수행하고, 양방향 물류의 각운동량을 재순환시키는 자립형 투석 플랫폼. 소행성 현지 자원으로 건설.**

**위치:** 헝가리아군 소행성대 (~1.8–2.0 AU)

**목적:** 소행성간 화물을 수신하고, 화성 방향(또는 화성 스윙바이 후 지구 방향)으로 재투척하는 양방향 물류 허브. 본질적으로 기계적 스윙바이 스테이션.

---

## 1. 왜 Mass Driver가 아닌가

### 1.1. Mass Driver의 피크 전력 문제

전자기 투석기(Mass Driver)는 소행성 표면에 1~2 km 트랙을 설치하고 전자기 유도로 Fe-Ni 빌렛을 가속한다. 문제는 **피크 전력**이다.

```
중간 벨트(2.7 AU) 기준, 빌렛 1톤:
  사출 속도: 4,790 m/s
  가속도: 1,000g (9,810 m/s²)
  가속 시간: v/a = 4790/9810 = 0.49초
  에너지: ½mv² = 0.5 × 1000 × 4790² = 11.47 GJ
  피크 전력: 11.47 GJ / 0.49초 = **23.4 GW**

→ GW급 에너지 저장 장치(커패시터 뱅크, SMES, 플라이휠)가 필수
→ 연속 충전 전력: 11.47 GJ / 156초(발사 간격) = 73.5 MW
```

23 GW 피크는 기존 가속기 기술로 원리상 가능하지만, **소행성 현지에서 건설**해야 하는 에너지 저장 인프라가 압도적으로 크다.

### 1.2. 스윙스테이션의 핵심 차이

스윙스테이션은 구조물 자체가 거대한 플라이휠이다. **수 분에 걸쳐** 천천히 스핀업하므로 피크 전력이 존재하지 않는다.

| | Mass Driver | 스윙스테이션 |
|--|-------------------|-----------------|
| 최대 속도 | 제한 없음 | ~960 m/s (구조 한계) |
| 연속 전력 | 74 MW | **~1 MW** |
| 피크 전력 | **23.4 GW** (0.5초) | **없음** (연속 가속) |
| 에너지 저장 장치 | GJ급 커패시터/SMES 필수 | **불필요** (구조물 자체가 플라이휠) |
| 건설 규모 | 트랙 0.4~1.2 km + 충전 인프라 | 쌍 로터 (암 500 m × 2) + 비회전 허브 |
| 소재 | Fe-Ni (트랙) | 소행성 Fe-Ni 자연 합금 (Co/P/C 열처리) |
| 에너지 재순환 | 불가 | 수신 시 운동에너지 → 회전에너지 회수 |

**핵심 차이: 피크 전력 제거 + 에너지 재순환.** GW급 에너지 저장 장치가 완전히 제거되며, 양방향 물류에서 에너지가 자립 순환한다.

> 헝가리아군에서 화성까지 Δv₁ = 930 m/s. 스윙스테이션 960 m/s로 전량 분담 가능.
> Δv 분담 근거 → [`전이궤도항법.md`](전이궤도항법.md) §2 참조

## 2. 구조: 쌍 로터 (Counter-Rotating Dual Rotor)

### 2.1. 전체 형상

두 개의 로터(A, B)가 중앙축을 공유하며 **서로 반대 방향으로 회전**하는 쌍 로터 구조. 각 로터는 수십 개의 암이 방사상으로 뻗어나가는 대형 회전체이며, 팁 속도 약 960 m/s. 시각적으로는 두 개의 거대한 바람개비가 서로 반대로 돌면서 중심축을 공유하는 모습이다.

```
            로터 A (시계 방향)
              ╲  │  ╱
               ╲ │ ╱
        ────────[허브]────────    ← 비회전 중앙부 (순 각운동량 ≈ 0)
               ╱ │ ╲
              ╱  │  ╲
            로터 B (반시계 방향)
```

| 구성 | 설명 |
|------|------|
| 로터 A | 시계 방향 회전, 화성 방향 발사/수신 |
| 로터 B | 반시계 방향 회전, 지구 방향 발사/수신 |
| 중앙 허브 | 비회전, 도킹 포트·태양전지·SMR·관제 시스템 |
| 중앙 모터 | 두 로터 사이 각운동량 재분배 |

### 2.2. 쌍 로터의 핵심 이점

**자이로스코픽 상쇄.** 단일 회전체는 거대한 자이로스코프로서 세차 운동이 발생하고, 회전면을 바꾸려면 막대한 토크가 필요하다. 반대 방향으로 도는 두 로터를 결합하면 **순 각운동량이 제로**. 외부에서 보면 시스템 전체가 비회전체처럼 행동하여 자세 제어가 극적으로 쉬워진다.

**각운동량 은행.** 로터 A가 화물을 던져서 각운동량을 잃으면, 중앙 모터가 로터 B에서 각운동량을 빌려와서 A를 보충한다. B가 화물을 받아서 각운동량이 과잉이면, 모터가 그걸 빼서 A에 넣는다. 모터는 각운동량을 "만드는" 게 아니라 **"재분배"만** 하므로 에너지 소모가 최소.

**양방향 동시 운용.** A가 시계 방향으로 돌면서 화성 방향으로 던지고, B가 반시계 방향으로 돌면서 지구 방향으로 던지면 — 하나의 스윙스테이션에서 **두 방향 항로를 동시 서비스**. 단일 로터는 한 번에 한 방향밖에 못 던지지만, 쌍 로터는 반대 방향 두 항로를 동시 처리한다.

**비회전 허브.** 두 로터를 잇는 중앙축은 순 각운동량 제로라서 사실상 정지해 있다. 외부에서 접근하는 화물선이 도킹하기에 이상적이며, 태양전지 어레이·SMR·관제 시스템을 전부 이곳에 배치한다. 도킹 후 어느 로터로 배정할지 결정해서 보내면 된다.

**반작용 토크 상쇄.** 단일 로터는 화물 발사 시 반작용 토크를 구조물 전체가 버텨야 하지만, 쌍 로터는 반작용이 상대 로터로 흡수되어 중앙축에 걸리는 순 토크가 상쇄된다.

### 2.3. 삼각격자 트러스

각 로터의 암은 **소행성 Fe-Ni 자연 합금**으로 구성된 삼각격자(isogrid) 트러스. 외부 합금 원소 반입 불필요 — 소행성 Fe-Ni에 자연 포함된 Co/P/C를 조성 제어 + 열처리로 활용.

- 삼각격자는 모든 부재가 순수 인장을 받아 합금의 인장강도를 가장 효율적으로 활용
- **단일 소재** → 전자빔 용접으로 전 접합부 제작 가능, 이종 금속 계면 없음
- 소행성 현지에서 제련→주조→인발→열처리 일관 공정

### 2.4. 구배 설계

허브 근처는 격자가 촘촘하고 부재가 굵으며, 끝으로 갈수록 성기고 가늘어지는 구배 설계. 960 m/s 팁 속도에서의 원심 응력을 허용 범위 내로 유지.

## 3. 소재 한계: 최대 팁 속도

원심 구조물의 최대 팁 속도는 소재의 **비강도(σ/ρ)**로 결정된다:

```
v_tip_max = √(2σ / ρ)

σ: 인장 강도 (Pa)
ρ: 밀도 (kg/m³)
```

이 공식은 암 길이와 무관하다. 암을 길게 만들면 ω가 낮아지지만, 원심력이 작용하는 질량이 늘어나서 상쇄된다.

### 소재별 팁 속도 비교표

| 소재 | 밀도 (kg/m³) | 인장 강도 (GPa) | **균일 단면 팁 속도** | 현지 제작 |
|------|-------------|---------------|---------------------|:---:|
| Fe-Ni 원광 (미가공) | ~7,874 | ~0.4 | 319 m/s | O |
| Fe-Ni 단순 열처리 (800 MPa) | ~7,874 | ~0.8 | 450 m/s | O |
| **Fe-Ni 자연 합금 (Co/P/C 열처리)** | **~7,874** | **~0.8–1.2** | **450–552 m/s** | **O** |
| Fe-Ni-Ti 변형 마레이징강 | ~7,900 | ~2.0 | 712 m/s | △ (달 Ti) |
| 마레이징강 (Fe-Ni-Co-Mo) | ~8,000 | ~2.5 | 791 m/s | △ (Co/Mo 부족) |
| 탄소섬유 (T700) | ~1,600 | ~3.5 | 2,092 m/s | X |

탄소섬유는 비강도가 압도적이지만 소행성 현지에서 제작이 불가능하다.

### 960 m/s 도달 논거: 테이퍼드 트러스

균일 단면 팁 속도는 소재의 이론 한계이지만, 실제 스윙스테이션은 **테이퍼드 구배 설계** — 허브에서 굵고 끝으로 가늘어지는 구조다. 테이퍼 비율(허브 단면적 / 팁 단면적):

```
A(hub)/A(tip) = exp(v²ρ / 2σ)

v = 960 m/s, ρ = 7,874 kg/m³

σ =  800 MPa: exp(4.54) ≈ 93:1
σ = 1,000 MPa: exp(3.63) ≈ 38:1
σ = 1,200 MPa: exp(3.03) ≈ 21:1
```

소행성 Fe-Ni 자연 합금 800~1,200 MPa 범위에서 테이퍼 비율 **93:1 ~ 21:1**. 삼각격자 트러스는 이 구배를 격자 밀도와 부재 굵기로 자연스럽게 구현한다. 1,000 MPa 이상 달성 시 38:1 이하로 현실적 범위에 진입.

**외부 자원 없이 960 m/s 달성이 가능하다.**

## 4. 소행성 Fe-Ni: 자연이 미리 합금해놓은 고성능 소재

### 4.1. 핵심 전환

소행성 Fe-Ni를 "순수 철-니켈 + 불순물"이 아니라, **자연 합금 상태의 고성능 소재**로 본다. 철 운석의 전형적 조성([Scottetal.2020](https://ui.adsabs.harvard.edu/abs/2020oeps.book..206S/abstract))에는 강화에 필요한 원소가 이미 최적 범위로 포함되어 있다. 외부에서 아무것도 가져올 필요 없이, 조성 제어 + 열처리만으로 고강도 합금이 된다.

### 4.2. 자연 포함 강화 원소

| 원소 | 함량 | 강화 메커니즘 | 기여 강도 |
|------|------|-------------|----------|
| **Ni** | 5~10% | 기지 고용 강화, 인성 향상 | 기본 |
| **Co** | 0.4~1.0% | 고용 강화, 마르텐사이트 변태 온도 상승, Ni₃Co 석출 촉진 | +50~80 MPa |
| **P** | 0.1~0.5% | 고용 강화 (0.1%당 ~60 MPa), 슈라이버사이트(Fe,Ni)₃P 석출 강화 | +150~200 MPa |
| **C** | 미량~0.4% | 코헤나이트(Fe,Ni,Co)₃C 탄화물 강화, 마르텐사이트 변태 | +200~400 MPa |

### 4.3. 각 원소 상세

**코발트 (Co) — "공짜 합금 원소"**

지상 마레이징강(18Ni-250급)의 Co 8~12%에 비하면 0.5%는 소량이지만, 마르텐사이트 변태 온도를 올려주고 Ni 금속간 화합물 석출을 촉진한다. 별도 첨가 없이 열처리만으로 강화에 기여.

**인 (P) — "숨은 보석"**

지상 철강에서 P > 0.05%는 제거 대상이지만, 이는 연성이 필요한 구조물 기준이다. 스윙스테이션 삼각격자는 **모든 부재가 순수 인장**을 받으므로, 약간의 연성 저하를 감수하고 강도를 취하는 트레이드오프가 성립. 운석의 0.1~0.5%는 강화 극대화 + 취화 직전의 최적 구간. 슈라이버사이트(Fe,Ni)₃P를 열처리로 기지에 균일 재분배하면 석출 강화까지 추가.

**탄소 (C) — "사실상 강철"**

Campo del Cielo 운석 기준 C ~0.37%([WGBH](https://lsintspl3.wgbh.org/en-us/chemical-analysis/)). Fe-Ni에 C 0.2~0.4%가 들어가면 코헤나이트(Fe,Ni,Co)₃C가 형성 — 지상 강철의 시멘타이트와 본질적으로 동일하되 Ni/Co 치환으로 더 강하다. 퀜칭 + 템퍼링으로 마르텐사이트 조직을 만들면 강도가 극적으로 상승.

### 4.4. 합산 강도 추산

```
기본 Fe-8Ni:                     400~500 MPa
+ Co 0.5% 고용 강화:             +50~80 MPa
+ P 0.3% 고용 + 석출 강화:       +150~200 MPa
+ C 0.3% 탄화물 + 마르텐사이트:  +200~400 MPa
───────────────────────────────────────────
합계:                            800~1,200 MPa
```

### 4.5. 열처리 공정

```
1. 용융: 소행성 Fe-Ni 원광을 전기로/태양로로 용해
2. 조성 제어: Co/P/C 비율을 목표 범위로 조정 (제거가 아닌 배분)
3. 주조·인발: 규격 봉재로 성형
4. 오스테나이트화: 900~1,000°C 가열 → 균질화
5. 퀜칭: 급랭으로 마르텐사이트 변태 유도
6. 템퍼링: 400~500°C에서 시효 → 탄화물·인화물 석출 최적화
```

모든 온도가 전기로 달성 범위. 진공 환경이 열처리에 오히려 유리(산화 방지).

### 4.6. 원소 조달 — 완전 자립

| 원소 | 함량 | 공급원 | 비고 |
|------|------|--------|------|
| Fe | ~88% | 소행성 (M-type) | 주성분 |
| Ni | 5~10% | 소행성 (M-type) | 자연 포함 |
| Co | 0.4~1.0% | 소행성 (M-type) | 자연 포함 |
| P | 0.1~0.5% | 소행성 — 슈라이버사이트 | 자연 포함 |
| C | 미량~0.4% | 소행성 — 코헤나이트 | 자연 포함 |

**지구에서도 달에서도 아무것도 가져올 필요 없다.** 필요한 건 용광로와 열처리 기술뿐이며, 이는 수천 년 된 인류 야금학의 영역이다.

### 4.7. 선택적 업그레이드 경로: 달 Ti

달 일메나이트(FeTiO₃)에서 Ti를 추출하여 Fe-Ni-Ti 변형 마레이징강(σ ~2.0 GPa)으로 업그레이드하는 경로가 존재한다. DABEL5 부트스트랩이 EML5에서 시작하므로 달 자원 접근이 전제에 포함. 단, 기본 설계가 자연 합금만으로 성립하므로, 이는 **필수가 아닌 보험**이다.

## 5. 에너지 원리

### 5.1. 쌍 로터 = 쌍 플라이휠

두 로터 각각이 거대한 플라이휠. 진공이므로 마찰 손실 제로.

```
초기 스핀업 에너지 (로터 1기, 만 톤급): ~820 GJ
쌍 로터 합계: ~1,640 GJ
→ 소행성에 앵커한 상태에서 수년에 걸쳐 수십 kW급으로 천천히 가속 후 분리
```

### 5.2. 각운동량 은행

쌍 로터의 핵심 에너지 메커니즘:

```
[로터 A] ←── 중앙 모터 ──→ [로터 B]
  화물 발사 → 각운동량 감소       각운동량 과잉 ← 화물 수신
       └── 모터가 B에서 빌려 A로 재분배 ──┘
```

- **발사 시**: 해당 로터의 각운동량 소실 → 중앙 모터가 상대 로터에서 보충
- **수신 시**: 해당 로터의 각운동량 과잉 → 중앙 모터가 상대 로터로 이전
- 모터는 각운동량을 **생성하지 않고 재분배만** 수행 → 에너지 소모 최소

### 5.3. 에너지 자립 구조

양방향 물류가 균형을 이루면 두 로터 사이에서 각운동량이 순환하며 외부 에너지 보충 없이 영구 운용 가능.

순 각운동량 차액이 쌓이는 경우(한쪽 방향으로만 화물이 많이 나갈 때) 태양전지나 SMR이 모터를 돌려 보충. 단, 이는 **차액분만** 채우면 되므로 전체 회전에너지를 처음부터 만드는 것과는 규모가 차원이 다르다.

## 7. 발사 시퀀스

1. 화물선이 **비회전 중앙 허브**의 도킹 포트에 접안
2. 목적지에 따라 로터 배정 (화성행 → 로터 A, 지구행 → 로터 B)
3. 배정된 로터의 암으로 진입, 원심력에 의해 자연스럽게 바깥으로 이동
4. **전자기 브레이크**로 이동 속도를 제어 (제동 에너지는 회수 가능)
5. 암 끝에 도달, 궤도역학적으로 목표 방향이 맞는 타이밍에 **래치 해제**
6. 접선 방향으로 방출, 960 m/s 초기속도 획득
7. 화물선 자체 SEP 엔진이 나머지 Δv를 채워 전이궤도 진입

### 양방향 동시 운용

로터 A가 화성 방향으로 던지는 동안 로터 B가 지구 방향으로 던질 수 있다. 단일 로터는 한 번에 한 방향밖에 못 던지지만, 쌍 로터는 **반대 방향 두 항로를 동시 처리**한다.

### 릴리즈 정밀도

```
릴리즈 타이밍 오차: ±0.1 ms
  → 회전 속도 ω = 1.92 rad/s
  → 방향 오차: ω × 0.0001 = 0.000192 rad = **±0.011°**
  → 속도 오차: ±수 m/s

보정: 화물선 SEP가 수일에 걸쳐 정밀 보정 → 스윙스테이션 정밀도 요구 크게 완화
```

### 반동 토크 상쇄

쌍 로터 구조에서 반작용 토크는 **상대 로터로 흡수**되어 중앙축에 걸리는 순 토크가 상쇄된다. 단일 로터에서 필요했던 카운터매스나 동시 발사 조율이 불필요.

- 발사 시 해당 로터의 각운동량 감소 → 중앙 모터가 상대 로터에서 즉시 재분배
- 양쪽 로터에서 동시 발사 시 반작용이 서로 상쇄 → 중앙축 부하 최소

## 8. 수신(포착) 시퀀스

1. 다른 소행성에서 날아오는 화물선이 접근, SEP로 대략적 속도 매칭
2. 암 끝의 **수백 m~수 km 길이 Fe-Ni 테더**(와이어 케이블)가 포착 역할
3. 테더 끝의 전자석 또는 후크가 화물선의 포착 링에 걸림
4. 암 끝 **윈치가 장력을 제어**하며 수십 초에 걸쳐 줄을 감아들임
5. 화물의 운동에너지가 구조물 회전에너지로 흡수
6. 화물선이 암 끝 레일에 안착, 허브 방향으로 이동 개시

### 테더 포착의 이점

- **포착 창 확대**: 타이밍 정밀도 요구 완화
- **충격 분산**: 줄의 탄성이 충격을 수 초에 걸쳐 분산 (낚시 드래그 릴 원리)
- **자연 동기화**: 윈치로 줄을 감아들이면 반경 감소 → 각운동량 보존으로 화물이 자연스럽게 회전 동기화

## 10. 추진 체계 분업

출발 슬링스테이션이 초기 Δv ~960 m/s를 제공하여 소행성 궤도 이탈 및 화성 전이궤도 진입 처리. 화성 플라이바이(δ≈134°)가 근일점을 1.8→1.35 AU로 무상 강하. 이후 화물선 자체 SEP(~2,300 m/s)가 근일점을 1.35→1.0 AU로 나선 감속. 달 플라이바이 3회(~3개월)로 지구 포획. 슬링스테이션이 초기 이탈 구간을 대신하므로 SEP 부담 경감.

중간 경유 스윙스테이션(자유 궤도)은 주로 **궤도면 전환·방향 미세 조정**에 사용. 헝가리아군 항로의 출발 가속은 소행성 표면 앵커링 슬링스테이션이 담당한다.

## 13. 스윙포트 (Swing Port): 클러스터 확장

단일 스윙스테이션에서 시작하여, 물류량 증가에 따라 같은 거점에 스테이션을 추가 배치하면 하나의 **거대 항만 단지 — 스윙포트**가 된다.

### 13.1. 전 방위 커버

12개 쌍 로터가 각각 **다른 회전면**으로 배치되면 전 방위 커버가 달성된다:

```
           경사 궤도용 (황도면 밖)
                ╱
  메인벨트 방향 ── [클러스터 중심] ── 화성 방향
                ╲
           내행성 방향 (지구·금성)
```

단일 쌍 로터는 회전면에 수직인 방향으로 던지고 받을 수 없다는 근본적 제약이 있다. 12쌍이 각기 다른 회전면으로 배치되면 이 제약이 사라지고, **발사 창 제약 자체가 소멸** — 어떤 방향에서 화물이 오든, 어떤 방향으로 보내든 적절한 회전면의 스테이션이 즉시 처리.

### 13.2. 처리량

```
쌍 로터 1기: 암 30개 × 2로터 = 60개 암
12쌍 클러스터: 720개 암 동시 운용

회전 주기 ~3.3초, 분당 수십 건 입출항 가능
시간당 수백 건, 하루 수천 건

컨테이너 100톤 기준:
  하루 처리량: 수십만 톤급
```

### 13.3. 내부 위계

클러스터 내부에 자연스럽게 위계가 형성된다:

| 계층 | 규모 | 팁 속도 | 역할 |
|------|------|---------|------|
| 대형 (외곽) | 암 500 m, Fe-Ni 자연 합금 (열처리) | 960 m/s | 장거리 고속 화물 (메인벨트·화성·지구) |
| 소형 (내부) | 암 100~200 m, 순수 Fe-Ni | 100~200 m/s | 클러스터 내 재배분, 단거리 셔틀 |

소형 스테이션은 팁 속도가 낮으므로 구조 요구가 극적으로 낮아진다. 열처리 불필요한 순수 Fe-Ni 합금으로도 충분하며, 건설 복잡도가 대형의 수분의 1.

인천공항에 국제선 터미널과 국내선 터미널이 있고 셔틀이 잇는 것처럼 — 메인벨트에서 960 m/s로 날아온 원석 컨테이너를 대형 스테이션이 받아 감속하고, 소형 스테이션이 저속으로 제련 모듈·조립 모듈·데이터센터 모듈 등 각 목적지로 분배.

### 13.4. 자기복제 성장

다이슨 패널 더블링이 진행될수록 물류량이 지수적으로 증가. 원석, 구조재, 교체 부품, 코팅 소재가 끊임없이 순환해야 하므로 스윙스테이션도 **자기복제로 따라 증가**:

```
Phase 1: 스윙스테이션 1기 (단일 쌍 로터)
Phase 2: 3~5기 클러스터 (주요 항로 커버)
Phase 3: 12기+ 스윙포트 (전 방위 커버)
Phase 4: 20~50기 메가포트 (다이슨 스웜 물류 규모)
```

스윙스테이션 자체가 소행성 Fe-Ni 자연 합금으로 구성되어 외부 자원 없이 현지에서 완전 자립 복제 가능. 더블링이 진행될수록 주요 요충지의 스테이션이 12기 → 20기 → 50기로 불어나며, 주변에 정비 시설·연료 보급소·창고·관제 센터가 붙고 사람이 상주하면서 **독자적 경제권**이 형성된다.

### 13.5. 스윙포트의 위상

스윙포트는 지구의 싱가포르나 로테르담처럼, 자체로는 자원을 생산하지 않지만 **물류 거점이라는 위치만으로 거대한 경제가 돌아가는 허브**다.

DABEL5 전체 구조에서의 위치:

```
[SEL5/EML5 제조 허브] ←→ [스윙포트 네트워크] ←→ [소행성대 채광 기지]
                              ↕
                      [다이슨 스웜 에너지]
```

스윙스테이션이 DABEL5의 순환계라면, 스윙포트는 DABEL5의 심장이다.

## 14. 스윙포트 네트워크

요충지마다 형성된 스윙포트들이 서로 던지고 받으면 **태양계 규모의 물류 네트워크**가 된다.

```
스윙포트 A (헝가리아) ←→ 스윙포트 B (화성 궤도)
        ↕                              ↕
스윙포트 C (중간 벨트)   ←→  스윙포트 D (지구 궤도)
```

- 화물이 순환할수록 각 포트의 회전에너지가 재순환되는 자립형 인프라
- 화물 입장에서는 각 포트에 들러 방향 전환과 가속을 공짜로 받는 **우주 고속도로**
