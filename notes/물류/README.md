## 읽는 순서

### 처음 읽을 때 (하향식)

```
전이궤도항법 ─── 궤도역학 전체 그림, Δv 체계, 로드맵
     │
     ▼
헝가리아-화성 항로 ─── 전 구간 운용, Phase별 상세
     │
     ├──▶ 슬링스테이션 ─── Phase 1 가속 (표면 앵커링, 단일 로터)
     │
     ├──▶ 스윙스테이션 ─── 편향·포착 (자유 궤도, 듀얼 로터)
     │
     └──▶ 화물선 ─── 추진 원리, 질량 예산
              │
              └──▶ WTAL-SEP ─── 스러스터 모듈 하드웨어
```

### 특정 주제만 참고할 때

- **궤도역학 이론이 궁금하다** → `전이궤도항법.md`
- **전 구간 운용을 알고 싶다** → `헝가리아-화성 항로.md`
- **투석기(가속) 설계가 궁금하다** → `슬링스테이션.md`
- **편향기(방향 전환) 설계가 궁금하다** → `스윙스테이션.md`
- **화물선 추진 원리가 궁금하다** → `화물선.md`
- **스러스터 상세가 궁금하다** → `WTAL-SEP.md`

## 교차 참조 네트워크

