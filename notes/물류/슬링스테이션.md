# 슬링스테이션 (Sling Station)

소행성 표면에 앵커링된 원심 투석기. 현지조달 Fe-Ni 강체 트러스 회전체로, 태양열 기계동력으로 회전에너지를 축적한 뒤 화물에 운동량을 전달하여 궤도 간 전이 속도를 부여한다.

---

## 핵심 원리

### 플라이휠 배터리

슬링스테이션은 본질적으로 **플라이휠 에너지 저장 장치**다. 태양열 집광 → 밀폐 브레이턴 터빈 → 기어트레인 → 로터 축 직결. 열에서 기계로 **한 단계** 변환. 전기 변환 경로(열 → 기계 → 전기 → 기계)의 중간 두 단계가 제거되어 전달 효율 98% 이상.

발사 간격 사이에 터빈이 수십 MW를 꾸준히 공급하면서 로터를 정격 회전수까지 올려놓는다(충전). 화물 적재 시 로터가 느려지고(방전), 터빈이 다시 복원하는 사이클을 반복한다.

### 매스 드라이버와의 비교

| | 매스 드라이버 | 슬링스테이션 |
|---|---|---|
| 에너지 저장 | 축전기 (전기) | 로터 질량 (기계) |
| 방출 | 전자기 변환 (GW급 순간) | 기계적 운동량 전달 |
| 변환 단계 | 4단계 (전기→축전→스위칭→코일) | 1단계 (열→기계) |
| 전체 효율 | 50~70% | 90%+ |
| 방향 전환 | 불가 (감속+재가속 2벌 필요) | 가능 (흡수→릴리스) |

### 릴리스는 에너지 비용 제로

회전체에서 화물을 풀어주면 접선 방향으로 비행한다. 이때 남은 구조물의 각속도 ω는 **변하지 않는다** — 관성모멘트 감소와 각운동량 감소의 비율이 동일하기 때문. 에너지 비용은 **다음 화물을 적재할 때** 발생한다. 정지 상태의 화물을 회전하는 팁에 올리면 ω가 떨어지고, 터빈이 복원한다.

### 반작용 관리

#### 스핀 토크

앵커링 단일 로터이므로 발사 반작용 토크를 소행성이 흡수한다. 단발 교란은 소행성 관성모멘트 대비 무시 가능. **듀얼 로터 불필수** — 자유 궤도 스윙스테이션과의 핵심 차이.

#### 궤도 반작용 — 소행성은 로켓이다

핵심 제약은 **선운동량 누적**이다. 지구행 화물은 역행(retrograde) 사출이므로 소행성은 순행 방향으로 밀린다. 소행성 궤도가 점진적으로 상승하며, 이 과정은 사출 속도를 배기 속도로 하는 로켓 방정식을 따른다:

```
Δv_asteroid = v_sling × ln( 1/(1−f) )
            = 960 × ln( 1/(1−f) )   m/s

f = 누적 사출 질량 / 초기 소행성 질량
```

순행 Δv가 누적되면 역행 960 m/s 사출로 도달 가능한 화물 근일점도 함께 올라간다:

| 사출 비율 f | Δv | 소행성 궤도 | 화물 근일점 | 화성 조우 |
|---|---|---|---|---|
| 10% | 101 m/s | 1.82 AU | 1.53 AU | 양호 |
| 20% | 214 m/s | 1.84 AU | 1.54 AU | 가능 |
| 30% | 343 m/s | 1.86 AU | 1.56 AU | 원일점 부근만 |
| 50% | 665 m/s | 1.91 AU | 1.60 AU | 사실상 불가 |

화성 원일점 1.666 AU. **실용 사출 한계: 소행성 질량의 ~20–30%.**

이것은 가속 방식(슬링, 매스 드라이버, 로켓)과 무관한 뉴턴 제3법칙의 귀결이다. 발라스트 사출(규산염도 Si·O₂·세라믹 용처가 있어 낭비)이나 SEP 보정(104 kN 연속 = 1.5 GW, 슬링 터빈의 30배)으로도 현실적으로 해결되지 않는다. 소행성의 선운동량 예산은 채굴의 근본 제약.

#### 설계 함의

- **채굴량 ≠ 사출량**: 현지 건설·제련·추가 슬링에 소비되는 질량은 궤도에 영향 없음
- **대형 소행성 우선**: 10 km급(~1,050억 t)의 20% = 210억 t — 단일 소행성으로 충분
- **소형은 순차 이동**: 2 km급(~33억 t)의 20% ≈ 6.6억 t 사출 후 다음 소행성으로
- Δv 누적은 결정론적 — 항법 계산에 소행성 현재 궤도 실시간 반영

## 기준 설계 스펙

### 구조

| 항목 | 값 | 비고 |
|---|---|---|
| **암 길이** | 1 km × 양쪽 | 직경 2 km |
| **팁 속도** | 960 m/s | 네트워크 표준 |
| **각속도** | 0.96 rad/s (9.17 RPM) | 주기 6.54초 |
| **팁 가속도** | 94 g (922 m/s²) | 벌크 화물 전용 |
| **재료** | Fe-Ni 마레이징강 (1,000 MPa) | Co/P/C 소행성 자연 조성 활용 |
| **테이퍼 비** | 38:1 (허브:팁 단면적) | 삼각격자 구배 설계 |
| **팁 트러스 외형** | 1.0 m × 1.0 m | 금속 단면적 0.094 m² |
| **허브 트러스 외형** | 6.0 m × 6.0 m | 금속 단면적 3.58 m² |

### 질량

| 항목 | 값 |
|---|---|
| 한쪽 암 | 13,052 톤 |
| 양쪽 암 합계 | 26,104 톤 |
| 허브 (추정) | 1,305 톤 |
| **스테이션 총 질량** | **27,409 톤** |
| 화물/스테이션 질량비 | 1:269 (0.37%) |

전량 소행성 현지조달 Fe-Ni. 티타늄 합금 노드(5~10%)만 외부 반입.

### 소재 강화

소행성 Fe-Ni 자연 조성 내 미량 원소를 그대로 활용한다. 별도 첨가 불필요.

| 원소 | 함량 | 기구 | 강도 기여 |
|---|---|---|---|
| 코발트 | 0.4~1.0% | 마르텐사이트 변태 촉진, 금속간 화합물 석출 | +50~80 MPa |
| 인 | 0.1~0.5% | 고용 강화 + 슈라이버사이트 석출 | +150~200 MPa |
| 탄소 | 0.2~0.4% | 코헤나이트 탄화물 + 마르텐사이트 | +200~400 MPa |
| **합산** | | | **800~1,200 MPa** |

기준 설계는 1,000 MPa. 800 MPa 대비 질량 1/3, 1,200 MPa 달성에 필요한 열처리 정밀도 부담 회피.

## 화물 규격

### 기본 단위: 100 + 2 톤

| 항목 | 값 | 비고 |
|---|---|---|
| 적재 화물 | 100 톤 | Fe-Ni 잉곳, 규산염, 얼음 등 벌크 |
| 오버헤드 | 2 톤 | 포착 링, 항법 패키지, 냉가스 스러스터, 비컨 |
| **총 사출 질량** | **102 톤** | |

### 단위 선정 근거

- **오버헤드 비율**: 2~3% (10톤 화물이면 20~30%로 비경제적)
- **포착 정밀도**: 질량 증가 시 스윙스테이션 포착 충격 증대 — 102톤이 상한선
- **에너지 재충전**: 50 MW 터빈으로 15.7분 — 운용 템포와 균형
- **각운동량 교란**: 스테이션 질량 대비 0.37% — 무시 가능 수준

지구 비유: TEU(20톤 트럭 제약)보다는 벌크선 화물창 1개에 대응. 초기 인프라 표준 100톤, 정밀 화물/승객용 50톤·10톤 소형 규격 추후 추가.

## 에너지 시스템

### 동력원: 태양열 집광 + 밀폐 브레이턴 터빈

| 항목 | 값 |
|---|---|
| 터빈 출력 | 50 MW |
| 사이클 | 밀폐 브레이턴 (He 또는 CO₂) |
| 냉각 | 진공 방열판 |
| 전달 | 기어트레인 → 로터 축 직결 |

### PTO(Power Take-Off) 전환 시스템

터빈 출력을 클러치로 전환하는 이중 모드 운용:

- **로터 충전 모드**: 화물 적재 후 ω 복원 — 터빈 → 기어트레인 → 로터 축
- **발전 모드**: 로터 정격 도달 후 — 터빈 → 발전기 → 기지 전력 / 전해조 / 제련

화물 텀이 길면 발전 시간이 길어져 **별도 발전소 없이** 기지 전력을 커버. 물류량 증가 시 로터 충전 비중 증가, 자연스럽게 전용 발전 설비 확장 시점과 일치.

집광 거울 + 터빈 한 세트 = **발사 동력 + 기지 전력 + 프로펠런트 생산** 통합 에너지 허브.

### 에너지 수치

| 항목 | 값 |
|---|---|
| 스테이션 회전에너지 (빈) | 1.56 TJ (433 MWh) |
| 화물 포함 회전에너지 | 1.65 TJ (459 MWh) |
| **1회 적재 에너지** | **47.0 GJ (13 MWh)** |

### 운용 타이밍 (50 MW 터빈)

| 항목 | 값 |
|---|---|
| 초기 스핀업 (1회성) | 8.7 시간 |

## 스윙스테이션과의 역할 분담

| | 슬링스테이션 | 스윙스테이션 |
|---|---|---|
| **위치** | 소행성 표면 앵커링 | 자유 궤도 비행 |
| **기능** | 가속 (0 → 960 m/s) | 방향 전환 (960 → 960, 각도 변경) |
| **에너지** | 47 GJ / 회 (풀 가속) | 수 GJ / 회 (편향각 의존) |
| **각운동량** | 소행성이 흡수 | 듀얼 로터 자기상쇄 |
| **듀얼 로터** | 불필수 (앵커 있음) | 필수 (자유 비행체) |
| **비유** | 철도 기관차 (출발역 가속) | 선로 전환기 (분기점 방향 전환) |

슬링스테이션이 무거운 일(가속)을 하고, 스윙스테이션은 가벼운 일(방향 미세 조정)만 담당. 에너지 부담의 분업.

## 확장 경로

### 초기 (1~2기)

- 단일 소행성에 슬링 1기 건설, 50 MW 터빈
- 일 수 발 ~ 수십 발, 연 수만 톤
- 발전 모드로 기지 전력 겸용

### 성장기 (4기 병렬)

- 동일 소행성 또는 인근 소행성에 4기 배치
- 연 1,000만 톤 이상
- 전용 발전 설비 분리

### 성숙기 (네트워크)

- 주요 채굴 소행성마다 슬링 클러스터
- 스윙스테이션 네트워크와 연동
- 지구행: 슬링 → 화성 플라이바이 → SEP → 달 플라이바이 → SEL5 (`헝가리아-화성 항로.md`)
- 소행성간 단거리: 슬링 → 스윙 릴레이

### 자기복제 루프

첫 슬링스테이션이 던진 Fe-Ni로 다음 슬링스테이션 건설. 3~4기 되면 루프가 닫힌다.
