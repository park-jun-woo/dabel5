# 5A. 감속 터그 아키텍처 (Braking Tug Architecture)

> **"터그의 본질은 '어디서 대기하고, 어디서 분사하느냐'에 있다."**

---

## 5A.1. 문제 정의: 터그 랑데부 사이클

§5.3에서 NEP 감속 터그의 기본 파라미터를 정의했으나, **터그가 화물을 인계받고 감속 완료 후 다음 화물을 인계받기까지의 왕복 사이클 Δv**가 명시되지 않았다. 이 사이클 비용이 터그 아키텍처 선택을 결정한다.

핵심 질문: **터그는 어디에서 대기하고, 어떻게 화물과 속도를 맞추며, 감속 후 어떻게 복귀하는가?**

---

## 5A.2. 아키텍처 A: 전이궤도 릴레이 (원안 보완)

터그가 지구-태양 원궤도(1 AU) 근처에서 다양한 위상으로 분포하며, 전이궤도 말단에서 화물과 자연스럽게 합류하는 방식.

### 운용 개념

```
터그 위치: 태양 원궤도 ~1 AU (지구와 유사한 궤도, 다양한 위상에 분산)
화물 도착: 전이궤도 근일점 1 AU 통과 시 v = 35.97 km/s (태양 기준)
원궤도 속도: v_circ = 29.78 km/s
속도 차이: 6.19 km/s
```

### 사이클

| 단계 | 기동 | Δv | 질량 기준 | 소요 시간 |
|------|------|-----|----------|----------|
| ① 가속 매칭 | 터그가 6.19 km/s 가속 → 전이궤도 속도 | **6.19 km/s** | 터그만 | 수주 (NEP) |
| ② 도킹 | 화물 포획 | ~0 | — | 수시간 |
| ③ 감속 | 터그+화물 6.19 km/s 감속 → 원궤도 | **6.19 km/s** | 터그+화물 | ~89일 (100N급) |
| ④ 분리 | 화물 분리, 터그 대기 | ~0 | — | — |
| **합계** | | **12.38 km/s** | — | ~4개월 |

### 문제점

터그 자체의 가속 매칭(①)에 6.19 km/s가 추가로 소모된다. 터그가 경량(~10톤)이라 절대 추진제량은 적지만, NEP Isp 3,000s 기준:

```
터그 가속 (10톤): MR = e^(6190/29430) = 1.234 → 추진제 2.34톤
화물 감속 (터그 10t + 화물 100t + 추진제): MR = 1.234 → 추진제 ~25.7톤
총 추진제: ~28톤 (화물의 28%)
```

원안(§6.1)의 23.4%보다 약 20% 증가한다. 치명적이지는 않으나, **원안이 터그 매칭 비용을 누락하고 있었다**는 점은 수정이 필요하다.

### 완화 방안

터그가 이전 임무의 감속 완료 위치(원궤도)에서 **바로 다음 화물 배치를 기다린다.** 다음 화물이 같은 전이궤도로 도착하므로, 터그는 근일점 통과 시점에 맞춰 다시 가속하면 된다. "복귀" 자체가 불필요 — 터그는 원궤도 위의 유동 자산이다.

## 5A.3. 아키텍처 B: 지구 오베르트 킥 (Earth Oberth Kick)

터그가 지구 고궤도(EML4/EML5 높이)의 타원궤도에서 대기하다가, **지구 근지점에서 집중 분사**하여 중력우물의 오베르트 효과로 필요 Δv를 절감하는 방식.

### 오베르트 효과의 원리

지구 중력우물에 들어오면 위치에너지가 운동에너지로 변환되어 속도가 빨라진다. 근지점(가장 깊은 곳)에서 분사하면, **같은 Δv로 더 큰 궤도에너지 변화**를 얻는다. 화물과 터그 모두 근지점에서 빨라지지만, 빠른 속도끼리의 차이는 느린 속도끼리의 차이보다 작다.

### 속도 매칭 계산

```
지구 근지점 고도 200 km → r_p = 6,578 km
지구 탈출 속도 at r_p: v_esc = √(2 × GM_E / r_p) = √(2 × 3.986e14 / 6.578e6) = 11,010 m/s

화물 (v∞ = 6,190 m/s, 쌍곡선 궤도):
  v_근지점 = √(v∞² + v_esc²) = √(6190² + 11010²) = 12,632 m/s

터그 (EML4 높이 타원궤도, 원지점 ~38만 km):
  v_근지점 ≈ √(v_esc² - v_원지점²) ≈ 10,920 m/s
  (정확한 값은 타원궤도 에너지에 따라 변동)

근지점에서의 속도 차이: 12,632 - 10,920 = **1,712 m/s**
```

태양 기준 6,190 m/s 차이가 지구 근지점에서 **1,712 m/s로 72% 감소**한다.

### 운용 사이클

| 단계 | 기동 | Δv | 질량 기준 | 소요 시간 |
|------|------|-----|----------|----------|
| ① 대기 | EML4 높이 타원궤도 공전 | 0 | — | — |
| ② 근지점 가속 | 근지점 통과 시 분사 → 탈출 → 화물 속도 매칭 | **1.71 km/s** | 터그만 | **수분** |
| ③ 도킹 | SOI 외곽에서 화물 포획 | ~0 | — | 수시간 |
| ④ 근지점 감속 | 함께 지구 SOI 재진입, 근지점에서 역분사 → 포획 | **1.71 km/s** | 터그+화물 | **수분** |
| ⑤ 궤도 조정 | 포획 타원궤도에서 EML4 높이로 미세 조정 | ~0.1 km/s | 터그 | 수일 |
| **합계** | | **~3.5 km/s** | — | 수주 |

### 핵심 제약: 고추력 필수

**오베르트 효과는 근지점 근처의 짧은 시간 안에 분사를 완료해야 작동한다.**

```
근지점 통과 시간 (유효 구간 ±1,000 km):
  v_근지점 ≈ 12 km/s → 통과 시간 ≈ 2,000/12 ≈ 167초 (~3분)

1.71 km/s를 3분 안에 내려면:
  가속도 = 1,710 / 167 = 10.2 m/s² → 화물 100톤 기준 추력 = 1,020 kN

→ NEP (100 N) 불가. NTP 또는 화학 추진 필수.
```

### 추진제 비교 (NTP, Isp 900s)

```
② 가속 (터그 10톤만):
  MR = e^(1710/8829) = 1.214 → 추진제 2.14톤

④ 감속 (터그 10톤 + 화물 100톤 + 추진제 잔여):
  초기 질량 = 110 + 2.14(잔여) + m_prop
  MR = e^(1710/8829) = 1.214 → 추진제 ≈ 23.6톤

총 추진제: ~25.7톤 (화물의 25.7%)
```

아키텍처 A (NEP, 12.38 km/s, Isp 3,000s)의 28%와 비교하면 **추진제가 오히려 약간 적다.** Δv는 1/3.5로 줄었지만 Isp도 1/3.3으로 줄어서 거의 상쇄되는 구조다. 그러나 **터그 하드웨어가 극적으로 단순해진다** — 원자로·전력변환·이온엔진·방열판이 모두 불필요.

## 5A.4. 아키텍처 C: 달 반복 플라이바이 (Lunar Multi-Flyby)

터그가 달 궤도에서 대기하면서 **반복 달 플라이바이로 속도를 점진적으로 축적**한 뒤, 지구 근지점에서 잔여 Δv만 분사하는 방식. 아키텍처 B의 확장형이다.

### 달 플라이바이 Δv 능력

```
달 공전 속도 (지구 기준): v_Moon = 1,022 m/s
달 근접 통과 시 최대 편향 Δv: ~2 × v_Moon × sin(δ/2)

근접 고도 100 km, v∞ = 1 km/s 기준:
  tan(δ/2) = GM_Moon / (r_p × v∞²)
            = 4.905e12 / (1.837e6 × 1e6)
            = 2.67
  δ/2 = 69.5°, δ = 139°
  Δv = 2 × 1.022 × sin(69.5°) = 1.91 km/s (1회 최대)

실용 값: 매회 ~0.5~1.2 km/s (진입 속도와 기하학에 따라 변동)
```

### 속도 축적 과정

터그가 달 궤도에서 출발하여 반복 플라이바이로 지구 중심 궤도의 에너지를 높인다:

```
1회: 달 플라이바이 → 지구 중심 속도 +0.8 km/s → 타원궤도 확대
     ↓ ~27일 (달 공전주기)
2회: 달 플라이바이 → +0.8 km/s → 타원궤도 더 확대
     ↓ ~27일
...반복...

8~10회 후: 지구 중심 속도 누적 ~5~6 km/s
→ 지구 근지점에서 잔여 ~1.2 km/s만 분사 → 화물 속도 매칭
```

### 운용 사이클 (v∞ = 6.19 km/s 기준)

| 단계 | 기동 | Δv | 소요 시간 |
|------|------|-----|----------|
| ① 대기 | 달 궤도 공전 | 0 | — |
| ② 속도 축적 | 반복 달 플라이바이 (8~10회) | **0** | **~7~9개월** |
| ③ 최종 매칭 | 지구 근지점에서 잔여 Δv 분사 | **~1.2 km/s** | 수분 |
| ④ 도킹 | SOI 외곽에서 화물 포획 | ~0 | 수시간 |
| ⑤ 감속 | 지구 근지점에서 역분사 | **~1.2 km/s** | 수분 |
| ⑥ 복귀 | 반복 달 플라이바이로 감속 → 달 궤도 재진입 | **0** | **~7~9개월** |
| **합계** | | **~2.4 km/s** | **~16~20개월** |

### 추진제 계산 (NTP, Isp 900s)

```
③ 가속 (터그 10톤만):
  MR = e^(1200/8829) = 1.145 → 추진제 1.45톤

⑤ 감속 (터그 10톤 + 화물 100톤):
  MR = 1.145 → 추진제 ~16.0톤

총 추진제: ~17.4톤 (화물의 17.4%)
```

### 화성 스윙바이 경유 시 (v∞ = 2 km/s)

화성 스윙바이로 잔여 초과 속도가 2 km/s로 줄어든 경우:

```
지구 근지점에서:
  v_근지점_화물 = √(2000² + 11010²) = 11,191 m/s
  v_근지점_터그 = 10,920 m/s
  차이 = **271 m/s**

달 플라이바이 2~3회로 거의 매칭 가능.
잔여 Δv: 수백 m/s → 냉가스 또는 소형 화학 추진으로 충분.
```

| 파라미터 | 직접 전이 (v∞=6.19) | 화성 경유 (v∞=2.0) |
|---------|-------------------|-------------------|
| 달 플라이바이 횟수 | 8~10회 | **2~3회** |
| 선행 준비 시간 | 7~9개월 | **2~3개월** |
| 잔여 근지점 Δv | ~1.2 km/s | **~0.27 km/s** |
| 총 추진제/화물 | 17.4% | **~4%** |
| 왕복 사이클 | ~18개월 | **~6개월** |

## 5A.5. 달 궤도 대기의 추가 이점

### 연료 보급 인프라

달 극지방의 영구음영 크레이터에 수빙이 확인되어 있다. 달 표면 → 달 궤도로의 Δv가 ~1.8 km/s에 불과하므로, 달 수빙 → 전기분해 → H₂/O₂ → 터그 급유라는 현지 보급 체인이 성립한다. 지구에서 추진제를 올리는 것(LEO Δv ~9.4 km/s)보다 1/5 비용.

### 정비 및 교체

달 궤도 스테이션에서 터그의 점검·수리·모듈 교체가 가능하다. NEP 원자로 기반 터그는 원격 정비가 사실상 불가능하지만, NTP/화학 추진 터그는 기계 부품이 단순하여 로봇 정비에 적합.

### SEL5와의 연계

달 궤도에서 화물을 포획한 후, 최종 목적지인 SEL5(지구 뒤 60°)까지의 위상 전이는 별도 수단(NEP 터그 소수, 또는 화물 자체 냉가스)으로 처리. 달 궤도 → SEL5 Δv는 ~0.7 km/s로 극소.

## 5A.7. 추천: 단계별 혼용

단일 아키텍처를 고집할 이유가 없다. 인프라 성숙도에 따라 혼용한다.

```
Phase 2 (2050~, 벨트 초기 진출):
  → 아키텍처 A (NEP 릴레이) 소수 운용
  → 터그 생산 역량이 제한적이므로 고성능·소수 전략
  → 화성 경유 필수 (v∞ = 2 km/s로 터그 부담 최소화)

Phase 3 (2060~, Mass Driver 건설 후):
  → 아키텍처 B (오베르트 NTP) 전환
  → 터그 단가 낮아지고 NTP 대량 생산 시작
  → 사이클 최단 (~2주), 터그 수 최소 (~77기)

Phase 4 (2070~, 완전 성숙):
  → 아키텍처 C (달 플라이바이) 대량 투입
  → 자기복제로 터그 수천 기 생산 가능
  → 추진제 최소 (~4%), 가장 경제적
  → B + C 혼용: 급한 화물은 B, 일반 화물은 C
```

## 5A.8. §6.1 질량 예산 수정안

원안(§6.1)의 Phase 3 감속 Δv에 터그 랑데부 비용을 포함한 수정:

| 단계 | Δv | 담당 | 추진제 | 비고 |
|------|-----|------|--------|------|
| Phase 1: 출발 | 4.79 km/s | Mass Driver | **0** | 전력만 소모 |
| Phase 2: 순항 보정 | 0.02~0.05 km/s | 빌렛 냉가스 | 극소 (~8%) | 부표 유도 |
| Phase 3: 감속 | 6.19 km/s | 터그 (아키텍처별) | **17~28%** | 아래 참조 |
| Phase 4: 위상 전이 | 0.68 km/s | NEP 또는 냉가스 | 2.3% | 5회전 전이 |
