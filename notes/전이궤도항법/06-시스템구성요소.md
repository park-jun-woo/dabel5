## 5. 시스템 구성 요소

### 5.1. 전자기 투석기 (Mass Driver)

소행성 표면에 설치하는 고정 인프라. 전자기 유도로 도전성 Fe-Ni 빌렛을 직접 가속한다.

| 파라미터 | 내측 벨트 (2.2 AU) | 중간 벨트 (2.7 AU) | 외측 벨트 (3.2 AU) |
|---------|-------------------|-------------------|-------------------|
| 사출 속도 | 4.20 km/s | 4.79 km/s | 5.16 km/s |
| 가속도 1,000g | 트랙 0.90 km | 트랙 1.17 km | 트랙 1.36 km |
| 가속도 500g | 트랙 1.80 km | 트랙 2.35 km | 트랙 2.71 km |
| 가속도 100g | 트랙 9.0 km | 트랙 11.7 km | 트랙 13.6 km |
| 에너지/톤 | 8.82 GJ | 11.47 GJ | 13.31 GJ |

**Fe-Ni 빌렛은 구조적으로 1,000g 이상의 가속을 견딘다** (인간이나 정밀 기기가 없으므로). 트랙 1~2 km면 충분하다.

**연간 처리량 200,000톤 기준 (중간 벨트):**

```
빌렛 1톤/개 × 매 2.6분 발사 (연속 가동):
  에너지/발사: 11.47 GJ
  발사 시간 (1,000g): v/a = 4790/9810 = 0.49초
  피크 전력: 11.47 GJ / 0.49초 = 23.4 GW (에너지 저장 장치에서 방전)
  충전 전력: 11.47 GJ / 156초 = 73.5 MW (연속 충전)

소요 전력: ~74 MW (연속)
공급 방식: 태양열 집광기 또는 복수 SMR
```

74 MW는 대형 태양열 집광기(1 AU 기준 ~10만 m² 수광면적)로 달성 가능하다. 2.7 AU에서는 태양 플럭스가 1/7.3로 감소하므로 집광면적 ~73만 m² (860m × 860m) 또는 SMR 복수 기로 공급한다.

**Mass Driver 반동 제어:**

Mass Driver가 빌렛을 궤도 역방향으로 사출하면, 반작용으로 소행성은 순방향 운동량을 받아 궤도가 미세하게 상승한다. 연속 발사 시 누적 효과를 확인한다:

```
연간 200,000톤 × 4,790 m/s = 9.58 × 10¹¹ N·s/년

직경 100 km M-type (밀도 4,000 kg/m³):
  질량 = 2.09 × 10¹⁸ kg
  Δv = 0.46 μm/s/년 → 100년 누적 0.046 mm/s ← 완전 무시

직경 10 km M-type:
  질량 = 2.09 × 10¹⁵ kg
  Δv = 0.46 mm/s/년 → 100년 누적 46 mm/s ← 무시 가능 (v_orbit ~18 km/s)
```

**수십 km 이상의 벨트 소행성에서는 반동이 사실상 제로다.** 만약 소형 천체(~수 km)에서 고처리량 발사를 장기 운용할 경우, 역방향으로 광석 폐석(tailings)을 사출하여 운동량을 상쇄하는 보정이 가능하다. 채광 과정에서 발생하는 비금속 잔재가 무추진제 반동 매스 역할을 한다.

**소행성 탈출 속도:**

소행성 벨트의 중형 M-type 천체(직경 ~100 km, 밀도 ~4,000 kg/m³)의 탈출 속도는 ~75 m/s에 불과하다. Mass Driver 사출 속도(4~5 km/s)의 1/60 수준으로, 중력이 장애가 되지 않는다.

### 5.2. 항법 부표 네트워크 (Navigation Buoy)

부표는 한 곳에 정지해 있는 "등대"가 아니라, **전이궤도를 주기적으로 순환하는 무인 관제 위성**이다.

| 파라미터 | 값 |
|---------|-----|
| 궤도 | 벨트~1 AU 전이 타원궤도 (주기 ~2~3년) |
| 질량 | 200~500 kg/기 |
| 탑재 장비 | 항성 추적기, 라이다, 근거리 통신, 태양전지, 냉가스 자세 제어 |
| 네트워크 규모 | 전이 경로당 10~20기 (상이한 위상에 배치) |
| 커버 범위 | 각 부표 반경 ~10만 km (근거리 통신 유효 거리) |
| 기대 수명 | 10~20년 (연료 소진 시 교체) |

**부표의 역할:**

1. **정밀 위치 측정**: 근거리(수만 km)에서 빌렛의 위치·속도를 광학/라이다로 측정. DSN 원격 측정(정밀도 ~수 km)보다 2~3차수 우수한 미터급 정밀도 제공.
2. **궤도 보정 명령**: 측정된 궤적과 설계 궤도의 오차를 계산, 빌렛의 냉가스 추력기에 보정 명령 전송.
3. **군집 관리**: 동시 비행 중인 수백~수천 개 빌렛의 상대 위치를 모니터링하여 충돌 회피.

**부표 궤도 섭동 문제:**

궤도 섭동은 **가속도**(힘/질량)로 작용하므로 질량과 무관하다. 오히려 가벼운 부표가 태양 복사압에 **더 취약**하다. 따라서:

- 부표 자체도 정밀 궤도 결정(OD)이 필요 → 항성 추적기 + 태양 센서 + 지구 DSN 정기 교정
- 냉가스 자세·궤도 제어 예산: ~50 m/s/년 (스테이션키핑)
- 10년 수명 기준 총 Δv 예산: ~500 m/s → 냉가스(Isp 60s) 질량비 1.57 → 추진제 비율 36%

### 5.3. NEP 감속 터그 (Braking Tug)

1 AU 접근 구간에서 화물을 인계받아 감속시키는 **재사용 전기 추진 터그**.

| 파라미터 | 값 |
|---------|-----|
| 추진 방식 | NEP (원자력 전기 추진) 또는 SEP (태양 전기 추진, 1 AU에서 효율적). 1 AU에서는 태양 복사 1,361 W/m²로 방열판 효율이 심우주 대비 저하되므로, 도착 구간에서 SEP 전환이 방열 질량을 줄이는 데 유리하다. |
| 추력기 | 홀 스러스터 / VASIMR / 이온 드라이브 |
| Isp | 3,000 s (v_e = 29,430 m/s) |
| 감속 Δv₂ | 5.15~6.99 km/s (벨트 구간별) |
| 추가 위상 전이 Δv₃ | ~0.68 km/s |
| 운용 방식 | 전이궤도 말단에서 빌렛과 랑데부 → 감속 → 분리 → 복귀 |
| 재사용 | 감속 완료 후 다음 화물 배치로 이동 (터그 자체는 경량) |

**터그 1기의 처리 주기 (중간 벨트, Δv₂ = 6.19 km/s):**

NEP 감속은 저추력·장시간 가동이다. 100톤 화물을 6.19 km/s 감속하는 데 필요한 시간:

```
추력 10 N, 화물+터그 질량 ~124톤 (화물 100톤 + 추진제 23.4톤 + 터그 건조질량):
  가속도 = 10 / 124,000 = 8.06 × 10⁻⁵ m/s²
  소요 시간 = 6,190 / 8.06e-5 = 7.68 × 10⁷ s ≈ 2.4년

추력 100 N 급 (대형 NEP):
  가속도 = 100 / 124,000 = 8.06 × 10⁻⁴ m/s²
  소요 시간 ≈ 89일
```

대형 NEP 터그(추력 100 N급)면 **89일**에 100톤 빌렛 배치를 감속할 수 있다. 연간 4회 운용으로 400톤/기·년.

200,000톤/년 처리를 위해 필요한 터그 수: **~500기**. 이는 대규모지만, 터그 자체가 SEL5 공장에서 자기복제 방식으로 생산되면 점진적 확장이 가능하다.

### 5.4. 화물 빌렛 (Cargo Billet)

| 파라미터 | 값 |
|---------|-----|
| 소재 | Fe-Ni 합금 (소행성 현지 1차 제련) |
| 단위 질량 | 1~50 톤/개 |
| 형상 | 원기둥 또는 직육면체 (Mass Driver 보어에 적합) |
| 탑재 장비 | 수신기, 냉가스 추력기 (3축), 반사 코너큐브 (부표용 피식별체) |
| 보정 Δv 예산 | 20~50 m/s |
| 냉가스 질량비 | ~3~8% (Isp 60s, 50 m/s 기준: e^(50/588) - 1 = 8.9%) |
| 단가 | 극소 (수신기 + 냉가스 탱크 + 노즐, 현지 제작) |

빌렛은 **진정한 의미에서 "멍청한 화물"**이다. 자율 항법 AI, 고성능 센서, 추진 엔진이 없다. 다만 완전한 수동체는 아니며, 부표 신호를 수신하여 **"언제, 어느 축으로, 얼마나 분사할지"를 실행하는 최소 로직(초소형 ASIC 수준)**은 탑재한다. 이 로직은 부표가 계산한 보정 벡터를 수신 → 디코딩 → 3축 냉가스 밸브 개폐 타이밍 생성에 국한되며, 자체 궤도 결정(OD) 능력은 없다.
