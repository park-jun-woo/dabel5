## 4A. 화성 스윙바이 변형 (Mars Gravity Assist Variant)

직접 전이(§4)의 도착 Δv₂(6.19 km/s)를 줄이기 위해, **화성 중력을 무추진제 감속에 활용**하는 강화 아키텍처다.

### 4A.1. 개념

1 AU로 직접 보내는 대신, **화성 궤도(1.52 AU)까지만** Mass Driver로 사출하고, 화성 중력이 궤도를 꺾어 1 AU로 보낸다. 화성은 벨트와 지구 사이에 위치하므로 경유지로 자연스럽다.

```
[직접 전이]
  소행성(2.7 AU) ──Δv₁ 4.79──▶ 전이궤도 ──▶ 1 AU (v_excess = 6.19 km/s)

[화성 경유]
  소행성(2.7 AU) ──Δv₁ 2.78──▶ 전이궤도 ──▶ 화성(1.52 AU) ──스윙바이──▶ 1 AU (v_excess ≈ 2 km/s)
                   42% 감소                    ~4 km/s 공짜 감속            68% 감소
```

### 4A.2. 1단계: 화성까지의 전이 (Mass Driver)

직접 전이(근일점 1.0 AU)가 아니라 화성 궤도(근일점 1.52 AU)를 목표로 사출한다:

```
직접 전이 (2.7 AU → 1.0 AU):
  전이궤도 a_t = (2.7 + 1.0)/2 = 1.85 AU
  v_t at 2.7 AU = 13.33 km/s
  Δv₁ = |18.12 - 13.33| = 4.79 km/s

화성 경유 (2.7 AU → 1.52 AU):
  전이궤도 a_t = (2.7 + 1.52)/2 = 2.11 AU
  v_t at 2.7 AU = √(GM × (2/2.7 - 1/2.11))
               = 29.78 × √(0.741 - 0.474)
               = 29.78 × √0.267
               = 15.39 km/s
  Δv₁ = |18.12 - 15.39| = 2.73 km/s
```

| | 직접 전이 | 화성 경유 | 절감 |
|--|---------|---------|------|
| Mass Driver 사출 속도 | 4.79 km/s | **2.73 km/s** | **43%** |
| 트랙 길이 (1,000g) | 1.17 km | **0.38 km** | **68%** |
| 에너지/톤 | 11.47 GJ | **3.73 GJ** | **67%** |
| 연속 전력 (200kt/년) | 74 MW | **24 MW** | **68%** |

Mass Driver 규모가 1/3로 줄어든다. 트랙 380 m, 24 MW — 초기 건설에 훨씬 현실적이다.

### 4A.3. 2단계: 화성 스윙바이

화물이 화성 궤도(1.52 AU)에 도달하면 화성의 중력장을 이용해 궤도를 편향시킨다. **추진제 소모 제로.**

```
화물 at 1.52 AU (전이궤도):
  v_cargo = √(GM × (2/1.52 - 1/2.11))
          = 29.78 × √(1.316 - 0.474)
          = 29.78 × √0.842
          = 27.32 km/s

  v_Mars = √(GM/1.52) = 29.78 / √1.52 = 24.15 km/s

  v_infinity (화성 대비 초과 속도) = |27.32 - 24.15| = 3.17 km/s
```

**화성 근접 통과 편향각 계산:**

```
GM_Mars = 4.283 × 10¹³ m³/s²
근접 통과 고도 200 km → r_p = 3,390 + 200 = 3,590 km

tan(δ/2) = GM_Mars / (r_p × v_inf²)
          = 4.283e13 / (3.59e6 × 3170²)
          = 4.283e13 / 3.608e13
          = 1.187

δ/2 = 49.9°
δ = 99.7° ← 거의 직각 편향

태양 기준 최대 Δv 변화량:
  = 2 × v_inf × sin(δ/2)
  = 2 × 3.17 × sin(49.9°)
  = 2 × 3.17 × 0.765
  = 4.85 km/s
```

**화성 스윙바이 한 번으로 최대 ~4.85 km/s의 궤도 변경이 가능하다.** 이는 도착 Δv₂(직접 전이 시 6.19 km/s)의 약 78%에 해당한다. 추진제 소모는 제로.

최적 편향 방향으로 화물을 근일점 ~1.0 AU, 속도 초과분 ~2 km/s 이하인 새 궤도에 투입하면:

| | 직접 전이 | 화성 스윙바이 후 | 절감 |
|--|---------|--------------|------|
| 1 AU 도착 초과 속도 | 6.19 km/s | **~2 km/s** | **68%** |
| NEP 추진제/화물 (Isp 3,000s) | 26.3% | **~8%** | **70%** |
| 200,000톤 기준 추진제 | 52,600 t | **~16,000 t** | **70%** |

### 4A.4. 3단계: 잔여 감속 및 SEL5 도착

화성이 ~4 km/s를 감당한 후 잔여 ~2 km/s + 위상 전이 0.68 km/s = **~2.7 km/s**를 NEP 터그가 처리한다. 직접 전이의 6.87 km/s에 비해 NEP 터그 부담이 60% 감소.

```
잔여 감속 Δv₂ ≈ 2 km/s + 위상 전이 0.68 km/s = 2.68 km/s
NEP (Isp 3,000s): MR = e^(2680/29430) = e^0.091 = 1.095
추진제/화물 = 9.5%
```

### 4A.5. 운용 제약

**1) 발사 창 제한**

화성이 올바른 위치에 있어야 스윙바이가 성립한다.

```
화성 시노딕 주기: ~2.14년 (화성-지구 정렬 주기)
유효 발사 창: 각 주기당 ~2~4개월
→ 상시 발사 불가, 배치(batch) 발사로 전환
```

단, 하나의 발사 창에서 수개월간 연속 발사하면 수만 톤 처리가 가능하다. Mass Driver는 매 2.6분 발사이므로, 3개월 창에서 ~50,000톤 처리. 연간 2회 창이면 ~100,000톤/년.

**2) 궤도 정밀도 요구**

스윙바이는 근접 통과 고도에 극도로 민감하다:

```
고도 오차 ±10 km → 편향각 오차 ±0.5° → 출구 궤도 1 AU에서 ~수만 km 편차
```

부표 네트워크가 화성 접근 수주 전부터 정밀 궤도 결정(OD)을 수행하고, 빌렛의 냉가스 추력기로 최종 조준 보정을 실행한다. **부표의 가치가 스윙바이에서 극대화된다** — 직접 전이에서는 "있으면 좋은" 수준이지만, 스윙바이에서는 필수 인프라다.

특히 부표는 **화성 영향권(SOI) 진입 전 ~100만 km 구간에 집중 배치**해야 한다. 근접 통과 고도 오차는 SOI 진입 시점의 미세 오차가 근일점에서 기하급수적으로 증폭되는 구조이므로, 최종 보정 기회는 SOI 진입 직전이다. 이 구간에 3~5기의 부표를 수만 km 간격으로 배치하여 연속 OD → 보정 명령 → 실행의 폐루프를 구성한다.

**3) 지구 스윙바이 추가 가능성**

화성 스윙바이 후에도 잔여 초과 속도가 있으면 지구 스윙바이로 추가 감속이 가능하다:

```
지구 근접 통과 (v_inf = 2 km/s, r_p = 6,578 km):
  tan(δ/2) = GM_Earth/(r_p × v_inf²)
            = 3.986e14 / (6.578e6 × 2000²) = 15.14
  δ ≈ 170° (거의 완전 반전)
  → 2 km/s 잔여분 완전 제거 가능
```

**단, 지구 스윙바이에는 안전 문제가 있다.** 수만 톤의 Fe-Ni 빌렛을 지구 근접 궤도로 보내는 것은 행성 방어 관점에서 리스크다. 냉가스 보정 실패 시 빌렛이 지구에 충돌할 수 있다.

안전 운용 원칙:
- **화성 스윙바이만 사용** (기본): 지구 근접 없이 잔여 Δv는 NEP 터그가 처리
- **지구 스윙바이** (옵션): NEP 터그가 도킹 후 정밀 제어 하에서만 실행. "멍청한 화물" 단독으로는 절대 지구 근접 불가.

### 4A.6. 목성 스윙바이는?

목성은 질량이 지구의 318배로 편향 능력이 압도적이지만, 벨트 → 1 AU 수송에는 **부적합**하다:

```
벨트(2.7 AU) → 목성(5.2 AU): 전이궤도 a_t = 3.95 AU
  Δv₁ = 2.70 km/s (궤도 바깥쪽으로 가속 — 방향이 반대!)
  전이 시간: ~3.9년 (편도)

왕복 경로: 벨트 → 목성 → 1 AU = ~6년+ (직접 전이 1.3년 대비 5배)
```

목성은 벨트 바깥쪽에 있다. 1 AU로 가려면 안쪽으로 가야 하는데, 목성 경유는 바깥으로 돌아가는 셈이다. 수년의 추가 비행 시간만 들고 이점이 없다. (목성 스윙바이가 유용한 건 외행성 탐사에서 가속이 필요할 때다.)

### 4A.7. 종합 비교

| 아키텍처 | MD 사출 Δv₁ | 스윙바이 Δv | NEP Δv₂ | 추진제/화물 | 전이 시간 | 발사 창 |
|---------|-----------|-----------|--------|----------|---------|--------|
| 직접 MD+NEP | 4.79 km/s | — | 6.87 km/s | **26.3%** | ~1.3년+NEP | 상시 |
| **Mars 경유** | **2.73 km/s** | **~4 km/s** | **~2.7 km/s** | **~9.5%** | ~1.5년+NEP | ~2년 주기 |
| Mars+Earth | 2.73 km/s | ~6 km/s | ~0.7 km/s | ~2.5% | ~1.5년+NEP | ~2년 주기 |

**화성 스윙바이는 Mass Driver 규모 1/3, NEP 추진제 1/3로 줄인다.** 대가는 발사 창이 ~2년 주기로 제한되는 것인데, 배치 발사로 연간 ~10만 톤 처리는 충분히 가능하다.
