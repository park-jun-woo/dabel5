# 헝가리아군 경유 항로 통합 항법 설계 (Hungaria Route Integrated Navigation)

> **"현지 철로 만든 투석기가 던지고, 화물선의 전기 엔진이 다듬고, 화성이 꺾고, 달이 잡는다."**

---

## 1. 항로 개요

소행성 벨트 최내측 헝가리아군(~1.8 AU)에서 출발하여, 화성 중력 어시스트와 달 플라이바이 포획을 거쳐 SEL5에 도달하는 최소 인프라 항로다.

**자력 추진 화물선**: W-TAL 잡탕 홀 스러스터를 탑재하여 투석기 이후 모든 Δv를 자체 처리한다. Fe-Ni 화물 자체가 추진제이므로 별도 추진제 공급이 불필요하다.

```
[헝가리아군 1.8 AU]
  │
  ├─ Fe-Ni 원심 투석기 (750 m/s, 추진제 0, 전력 ~1 MW)
  │
  ▼
[전이궤도 → 화성 1.52 AU] ···· 잡탕 홀 자력 가속 180 m/s + 순항 보정 50 m/s
  │
  ├─ 화성 스윙바이 (δ ≈ 170°, Δv ~2 km/s, 추진제 0)
  │
  ▼
[전이궤도 → 1 AU] ···· 잡탕 홀 자력 감속 800 m/s
  │
  ├─ 달 플라이바이 포획 (1~2회, 추진제 ~0)
  │
  ▼
[지구 근방 궤도]
  │
  ├─ 추진 모듈 분리 → 달 궤도 스테이션으로 회수
  ├─ 화물 → SEL5 위상 전이 (680 m/s, 잔여 잡탕 홀 또는 냉가스)
  │
  ▼
[SEL5 도착]
```

## 2. 전 구간 Δv 분담표

| Phase | 구간 | Δv | 담당 | 추진 방식 | 추진제 |
|-------|------|-----|------|----------|--------|
| 1 | 헝가리아군 출발 | 750 m/s | Fe-Ni 원심 투석기 | 원심력 | **0** |
| 2 | 잔여 출발 가속 (→화성) | 180 m/s | 화물선 잡탕 홀 | 전기 추진 | ※ |
| 3 | 순항 보정 | 50 m/s | 화물선 잡탕 홀 | 전기 추진 | ※ |

| 5 | 도착 감속 | 800 m/s | 화물선 잡탕 홀 | 전기 추진 | ※ |
| 6 | 달 플라이바이 포획 | ~0 m/s | 달 중력 | 중력 어시스트 | **0** |
| 7 | SEL5 위상 전이 | 680 m/s | 화물선 잡탕 홀 | 전기 추진 | ※ |
| | **합계** | **~4,460 m/s** | — | — | — |
| | 화물선 자력 Δv 합계 (※) | **1,710 m/s** | 잡탕 홀 (Fe-Ni) | — | **10.2%** |
| | 무추진제 Δv 합계 | **2,750 m/s** | 투석기+중력 | — | **0** |

잡탕 홀 스러스터(Fe-Ni, Isp 1,800 s)로 1,710 m/s를 처리할 때:

```
MR = e^(1710 / 17658) = 1.102
추진제/화물 = 10.2%
500톤 화물 기준: 51톤 Fe-Ni (화물에서 분리)
```

## 3. 추진: 화물이 곧 추진제

> **추진 시스템 상세 → [`화물선.md`](화물선.md) 참조**

잡탕 금속 증기 홀 스러스터(W-TAL 하이브리드)는 **Fe-Ni 화물 자체를 추진제로 사용**한다. 화물의 10.2%를 용광로에서 증발·이온화하여 배기한다. 별도의 추진제 선택·조달·저장이 불필요하다.

| 항목 | 값 |
|------|-----|
| 추진제 | Fe-Ni 원광 (화물에서 분리) |
| 스러스터 | W-TAL 하이브리드 (자기 차폐) |
| Isp | 1,800 s (보수적, 300V, 55%) |
| 추진제/화물 | 10.2% (Δv 1,710 m/s) |
| 추진제 공급 체인 | **없음** |
| 유일한 별도 조달 | 냉가스 RCS용 CO₂ (화물의 0.8%) |

### 4.4. 전원: 태양전지 vs 원자력

| | 태양전지 (SEP) | 원자력 (NEP) |
|--|--------------|-------------|
| 1.8 AU 출력 (100 kW 기준) | ~60 kW (거리 제곱 감쇄) | 100 kW (일정) |
| 질량 | ~3톤 (패널) | ~8톤 (원자로+방열) |
| 현지 제작 | 어려움 (반도체) | 어려움 (핵연료) |
| 1 AU 성능 | **100 kW** (최적) | 100 kW (방열 부담 증가) |
| 추천 | **Phase 2~3** (단순, 경량) | Phase 4+ (대전력 필요 시) |

이 항로는 1.8 AU(출발) → 1.52 AU(화성) → 1.0 AU(도착)로 갈수록 태양에 가까워진다. 감속이 필요한 구간(1.5~1.0 AU)에서 태양전지 효율이 최대가 되므로 **SEP가 이 항로에 자연적으로 유리**하다.

### 4.5. 추력 시간 계산

```
화물선 총질량: ~520톤
추력: 10 N (SEP 100 kW급 잡탕 홀)
가속도: 10 / 520,000 = 1.92 × 10⁻⁵ m/s²

각 Phase별 소요 시간:
  Phase 2 (180 m/s): 180 / 1.92e-5 = 9.38e6 s ≈ 109일
  Phase 3 (50 m/s):  50 / 1.92e-5 = 2.60e6 s ≈ 30일
  Phase 5 (800 m/s): 800 / 1.92e-5 = 4.17e7 s ≈ 483일
  Phase 7 (680 m/s): 680 / 1.92e-5 = 3.54e7 s ≈ 410일

총 추력 시간: ~1,032일 ≈ 2.8년
```

추력 시간이 길지만, 어차피 궤도 전이 자체가 1.8년(헝가리아군→화성→1 AU) + 위상 전이 4.8년이다. 저추력 가속/감속은 탄도 비행과 병행되므로 **추가 시간이 거의 없다**. 감속 483일은 1 AU 접근 전 수개월간 역분사하는 것으로, 전이 시간 안에 흡수된다.

## 5. Phase별 상세 항법

### Phase 1: Fe-Ni 원심 투석기 발사 (헝가리아군 현지)

```
소행성 표면, 마레이징강 원심 투석기
  암 길이: 500 m, 팁 속도: 750 m/s
  스핀업: ~5분 (연속 ~1 MW)
  릴리즈 → 화물선이 소행성 궤도 역방향으로 사출

  v_소행성 = 22.20 km/s (1.8 AU 원궤도)
  v_사출 후 = 22.20 - 0.75 = 21.45 km/s
  → 근일점이 ~1.57 AU인 타원궤도 진입
  → 화성 궤도(1.52 AU)에 도달하려면 잔여 180 m/s 필요
```

발사 간격: 빌렛 배치 500톤 기준 약 1~2시간 (스핀업+릴리즈+재장전). 일일 12~24회, 발사 창 3개월에 ~50,000톤.

### Phase 2: 잔여 가속 (→ 화성행 전이궤도 확정)

```
투석기 릴리즈 직후, 화물선 잡탕 홀 점화
  Δv = 180 m/s (역방향, 추가 감속)
  → 근일점이 정확히 1.52 AU인 Hohmann 전이궤도 확정
  소요 시간: ~114일 (전이 초반에 완료)
```

이 단계에서 잡탕 홀은 투석기의 릴리즈 오차도 보정한다. 투석기의 릴리즈 타이밍 오차(±0.1 ms → 속도 오차 ±수 m/s, 방향 오차 ±0.01°)를 수일에 걸쳐 정밀 보정하므로, 투석기 자체의 정밀도 요구가 크게 완화된다.

### Phase 3: 순항 (헝가리아군 → 화성, ~1.07년)

```
전이궤도 비행: 1.8 AU → 1.52 AU
  전이 시간: ~1.07년

부표 네트워크가 궤도 결정(OD) 수행:
  → 화물선 위치·속도 측정 (미터급 정밀도)
  → 궤도 보정 명령 전송
  → 화물선 잡탕 홀가 보정 실행 (수일~수주)

순항 보정 Δv 예산: 50 m/s
  주로 화성 스윙바이 조준 보정에 사용
```

**부표의 역할:** 화물선은 자체 항법 센서(항성 추적기)를 탑재하므로 부표 의존도가 낮다. 부표는 보조적 정밀 OD 소스로, 배치 수를 최소화할 수 있다.

### Phase 4: 화성 스윙바이

```
화성 SOI 진입:
  v_infinity = 1,020 m/s (화성 대비)
  → 직접 전이 v∞(3,170 m/s)의 1/3

근접 통과:
  고도 ~200 km, r_p = 3,590 km
  편향각 δ ≈ 170° (거의 U-턴)

스윙바이 효과:
  태양 기준 Δv ≈ 2.03 km/s 궤도 변경
  → 출구 궤도: 근일점 ~1.04 AU
  → 1 AU 도착 시 초과 속도: ~800 m/s

정밀 조준:
  잡탕 홀이 SOI 진입 전 수주간 미세 보정
  → 근접 고도 오차 ±10 km 이내 달성
  → 자체 항법 센서 + 부표 보조
```

v∞가 1,020 m/s로 낮기 때문에 편향각이 170°에 달한다. 이는 출구 방향 선택의 자유도가 매우 높다는 의미로, 최적 궤도 설계가 용이하다. 다만 편향각이 클수록 근접 고도 민감도도 높아지므로 정밀 조준이 중요하다. 자체 조준 능력이 있어 부표 의존도가 낮지만, 화성 SOI 진입 전 ~100만 km 구간에 3~5기의 부표를 배치하면 안전성이 크게 향상된다.

### Phase 5: 자력 감속 (1.5 AU → 1 AU)

```
화성 스윙바이 출구에서 화물선은 근일점 ~1.04 AU 타원궤도에 진입.
1 AU 접근 수개월 전부터 잡탕 홀 역분사 시작.

  Δv = 800 m/s
  소요 시간: ~509일 (전이 비행과 병행)

  실제 운용: 화성 스윙바이 직후부터 연속 역분사
  → 전이 비행 중에 감속을 완료
  → 1 AU 도착 시 초과 속도 ≈ 0
```

### Phase 6: 달 플라이바이 포획 (지구 SOI)

```
화물선이 1 AU에서 잔여 초과 속도 ~0으로 접근.
잡탕 홀 감속이 완벽하면 포획 Δv ≈ 0이지만,
안전 마진으로 v∞ ~100~200 m/s 잔여를 가정.

달 1회 플라이바이:
  v∞ = 200 m/s, 근접 고도 100 km
  편향 Δv ≈ 400 m/s (200 m/s 잔여 완전 상쇄)
  → 지구 포획 타원궤도 진입

달 플라이바이 횟수: 1회 (충분)
소요 시간: ~27일 (달 공전주기 1회분)
```

Phase 5의 감속이 정상 완료되면 달 플라이바이는 **보험**이다. 감속이 불완전한 경우에만 필요하며, 정상 시에는 직접 포획궤도에 진입한다.

### Phase 7: SEL5 위상 전이

```
지구 포획궤도에서 SEL5(지구 뒤 60°)까지 위상 전이.

5회전 위상 전이 (§4 Phase 4 동일):
  Δv₁ = 0.34 km/s (안쪽 궤도 진입)
  Δv₂ = 0.34 km/s (원궤도 복귀)
  총 Δv = 0.68 km/s
  소요 시간: ~4.8년

화물선 잡탕 홀이 직접 수행 가능.
또는 추진 모듈을 달 궤도에서 분리하고,
화물은 최소 냉가스(0.68 km/s, Isp 160s → 추진제 ~4.5%)로 처리.
```

**추진 모듈 분리 시점의 트레이드오프:**

| 분리 시점 | 장점 | 단점 |
|---------|------|------|
| Phase 6 후 (달 궤도) | 모듈 빠른 회수, 재사용 주기 단축 | 위상 전이용 냉가스 추가 (화물의 4.5%) |
| Phase 7 후 (SEL5 도착) | 추진제 최소 (전구간 잡탕 홀) | 모듈 회수에 추가 비행 (SEL5→달 궤도) |

추천: **Phase 6 후 분리.** 모듈의 빠른 재사용이 우선이며, 위상 전이 4.5%의 냉가스 추가는 수용 가능하다.

② 화물선 프레임에 장착 → 헝가리아군으로 회항
   (달→헝가리아군 Δv ≈ 5.3 km/s, W-TAL 자력, ~6개월)
③ 헝가리아군 기지에서 새 화물 적재
   ·RCS 재충전 (CO₂)
   ·스러스터·음극 점검 (교체 불필요 — 4회 항차 ≪ 수명 한계)
④ 투석기 발사 → 화성 → 1 AU → 달 궤도 (~1.8년)
⑤ 추진 모듈 분리 → 달 궤도 스테이션 → 점검
⑥ ①로 복귀

1회 사이클: ~2.5년 (회항 6개월 + 전이 1.8년 + 정비 1개월)
모듈 수명 10년 기준: ~4회 왕복/모듈
모듈 수명 내 교체 부품: 없음 (정상 운용 시)
```

### 모듈 회항의 Δv

```
달 궤도 → 헝가리아군(1.8 AU):
  지구 SOI 탈출: ~1.0 km/s
  1 AU → 1.8 AU Hohmann 출발: ~3.4 km/s
  1.8 AU 도착 감속: ~0.9 km/s
  합계: ~5.3 km/s

모듈 단독(~10톤), Isp 1,800 s:
  MR = e^(5300/17658) = 1.351
  추진제 = 3.51톤 Fe-Ni (모듈의 35%)
```

모듈 회항 추진제는 달 궤도 스테이션에 비축된 Fe-Ni에서 공급한다. 이전 화물선이 배달한 Fe-Ni의 일부를 달 궤도에 비축하면 되므로, 별도 공급 체인이 불필요하다.

## 8. 전이 시간 종합

```
Phase 1:  투석기 발사                         0일
Phase 2:  잔여 가속 (전이 초반 병행)       (전이 중)
Phase 3:  순항 (1.8 AU → 1.52 AU)           ~1.07년
Phase 4:  화성 스윙바이                       ~수일
Phase 5:  감속 (전이 중 병행)             (전이 중)
          순항 (화성 → 1 AU)                 ~0.71년
Phase 6:  달 플라이바이                       ~27일
Phase 7:  SEL5 위상 전이                     ~4.8년
───────────────────────────────────────────
총 전이 시간:                               ~6.7년
(위상 전이 제외):                            ~1.9년

위상 전이(4.8년)를 단축하려면 Δv를 올리면 된다. 예를 들어 2회전 전이(Δv = 1.6 km/s)면 ~1.9년으로 줄어든다. 이 경우 총 전이 시간이 3.8년, 추진제가 ~9%로 증가.

## 10. 결론

헝가리아군 경유 항로는 DABEL5 아키텍처에서 **1986 DA(Phase 1)와 벨트 Mass Driver(Phase 3) 사이의 가교 단계(Phase 2)**를 담당한다.

핵심 이점:

1. **Mass Driver, GW급 에너지 저장이 불필요.** Fe-Ni 투석기(~1 MW)와 자력 추진 화물선만으로 운용.
2. **추진제 공급 체인 제로.** Fe-Ni 화물 자체가 추진제. 별도 채굴·가공·저장 불필요.
3. **지구 수입 의존 제로.** 투석기·프레임은 현지 Fe-Ni, 스러스터는 W 현지 회수.
4. **기술 실증도 최고.** 원심 가속, 화성 스윙바이, 달 플라이바이, 금속 홀 스러스터 모두 개별 실증 완료.
5. **화물 배달률 86.3%.** 추진제 10.2%로 1986 DA(14%)보다 효율적.

이 항로의 존재가 DABEL5의 전체 논리를 강화한다: 1986 DA에서 시간을 벌고, 헝가리아군에서 벨트 접근을 시작하며, 자기복제가 성숙하면 Mass Driver로 확장한다. 각 단계가 다음 단계의 인프라를 점진적으로 구축하는 사다리 구조다.

> **"현지 철로 만든 투석기가 던지고, 화물을 깎아 뱉으며 날고, 화성이 꺾고, 달이 잡는다. 우주에서 가장 비싼 건 지구에서 가져오는 것이다."**
