# DABEL5 심우주 물류: 소행성대 → SEL5 전이궤도항법 (Transfer Orbit Navigation)

> **"출발은 투석기가, 항해는 중력이, 감속은 전기가, 안내는 부표가 한다."**

---

## 1. 개요

소행성대(Main Belt, 2.2~3.2 AU)에서 채굴한 Fe-Ni 자원을 태양-지구 L5(SEL5)로 수송하는 **전이궤도항법** 물류 네트워크 설계안이다.

핵심 아키텍처 (2가지 변형):

```
[기본] 직접 전이:
[소행성]──Mass Driver──▶ 탄도 비행 ──▶ NEP 터그 감속 ──▶ [SEL5]
  출발 Δv₁               부표 유도          도착 Δv₂        위상 전이
  4.8 km/s               추진제: 0          6.2 km/s        Δv₃ 극소
  추진제: 0                                 추진제: 26%

[강화] 화성 스윙바이:
[소행성]──Mass Driver──▶ 탄도 비행 ──▶ 화성 ──▶ NEP 터그 ──▶ [SEL5]
  출발 Δv₁               부표 유도        스윙바이    잔여 감속     위상 전이
  2.8 km/s               추진제: 0       ~4 km/s 공짜  ~2 km/s    Δv₃ 극소
  추진제: 0                              추진제: 0    추진제: 8%
```

1986 DA 직접 수송(NTP/NEP 왕복)과 달리, **출발 Δv를 Mass Driver로 대체**하여 추진제를 절반으로 줄이고, 화물과 지능을 물리적으로 분리하여 비용을 극소화한다. 화성 스윙바이 경유 시 추진제를 추가로 2/3 절감할 수 있다.
