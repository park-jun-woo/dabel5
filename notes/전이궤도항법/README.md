# 전이궤도 항법 문서 구조

소행성 벨트 → SEL5 화물 수송 설계 문서. 원안(번호 시리즈)에서 헝가리아군 화물선 아키텍처로 진화.

---

## 문서 계보

```
[01-11] 원안: 빌렛+터그 아키텍처
   │
   ├─ [99-감속터그] §5 확장: 터그 3가지 형식 비교
   ├─ [99-헝가리아-화성-항로] §4 확장: 헝가리아군 발견
   │
   ├─ [화물선] ← 자력추진 화물선으로 전환 (터그 폐기)
   │     └─ [텅스텐TAL_NEP] ← NEP 모듈 상세 (W-TAL 스러스터)
   │
   └─ [헝가리아군_통합항법] ← 항로+화물선 통합 (최종 설계)
```

---
