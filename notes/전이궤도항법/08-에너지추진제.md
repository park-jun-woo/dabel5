## 7. 에너지 및 추진제 공급

### 7.1. Mass Driver 전력

| 소요 전력 | 공급 방식 | 비고 |
|----------|---------|------|
| ~74 MW (연속) | 태양열 집광기 (2.7 AU) | 수광면적 ~73만 m² (860m × 860m) |
| | 또는 SMR 복수기 (각 50~100 kW) | 740~1,480기 (확장 단계) |
| | 또는 SMR + 집광기 혼용 | 초기에 현실적 |

**Bootstrap 단계**에서는 처리량이 낮으므로(연간 수천 톤), 소형 SMR 수 기로 충분하다. 자기복제 확장에 따라 SMR과 집광기가 점진적으로 증설된다.

### 7.2. NEP 터그 추진제

NEP 터그의 추진제로는 다양한 선택지가 있다:

| 추진제 | 소스 | 장단점 |
|--------|------|--------|
| 크세논 (Xe) | 지구에서 공급 (초기) | 실증된 이온 추진제, 비쌈 |
| 아르곤 (Ar) | 지구/소행성에서 추출 | 저렴하나 추력 낮음 |
| **수소 (H₂)** | **C-type 소행성에서 물 전기분해** | 현지 조달 가능, VASIMR에 적합 |
| 비스무트 (Bi) | 소행성에서 추출 | 홀 스러스터에 적합, 저장 용이 |

**C-type 소행성**(벨트의 ~75%)은 10~20%의 물을 함유한다. 물 → 전기분해 → H₂(NEP 추진제) + O₂(산화제 또는 생명 유지). M-type 채광과 C-type 수자원 추출을 **병행 운용**하면 추진제의 현지 조달이 가능하다.

중간 벨트 기준 연간 추진제 소요량:
```
52,600톤 H₂ ← 473,400톤 H₂O (H₂ 질량분율 11.1%) ← C-type 소행성 ~237만~474만 톤 채굴
```
