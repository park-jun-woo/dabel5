## 8. 1986 DA 직접 수송 대비 비교

### 8.1. Δv 구조 비교

두 경로의 Δv는 총합뿐 아니라 **출발/도착 비대칭**이 본질적으로 다르다:

```
1986 DA:  출발 8.2 ████████▌  + 도착 1.1 █▏     = 9.3 km/s
벨트:     출발 4.8 ████▊     + 도착 6.2 ██████▏  = 11.0 km/s
                                                   ──────
          벨트가 출발 쉬움          1986 DA가 도착 쉬움
```

| 항목 | 1986 DA (NTP/NEP 직접) | 벨트 (전이궤도항법) |
|------|----------------------|-------------------|
| 출발 Δv₁ | **8.22 km/s** (편심궤도 고속) | **4.79 km/s** (원궤도 저속) |
| 도착 Δv₂ | **1.13 km/s** (거의 공짜) | **6.19 km/s** (감속 필수) |
| 총 Δv | ~9.3 km/s | ~11.0 km/s |
| 출발 해결 방식 | NTP/NEP 수송선 | **Mass Driver** (추진제 0) |
| 도착 해결 방식 | NTP로 충분 (추진제 14%) | **NEP 터그 필수** (추진제 23%) |
| 대규모 인프라 필요 | **아니오** | **예** (Mass Driver + NEP 터그 함대) |

### 8.2. 종합 비교

| 항목 | 1986 DA NTP/NEP 직접 수송 | 소행성 벨트 전이궤도항법 |
|------|------------------------|---------------------|
| 대상 | 단일 NEA (1986 DA) | 벨트 전체 (수백 M-type) |
| 자원량 | ~수십억 톤 (유한) | **~10¹⁹ kg+ (사실상 무한)** |
| 총 Δv | ~9.3 km/s | ~11.0 km/s |
| Δv 구조 | 출발 무겁고 도착 가벼움 | 출발 가볍고 도착 무거움 |
| 도착 감속 | NTP(Isp 900s)로 충분 | NEP(Isp 3,000s) 또는 Mass Driver 필수 |
| 추진제 (NTP) | 화물의 ~186% (출발+도착) | Mass Driver 분담 시 29.8% |
| 추진제 (NEP) | 화물의 ~37% | Mass Driver 분담 시 26.3% |
| Mass Driver | **불필요** | 필요 (1~2 km 트랙, 74 MW) |
| NEP 터그 | 왕복 수송선 (소수) | 감속 전용 터그 (~500기, 재사용) |
| 채광선 투입 | **쉬움** (2038 근접 0.21 AU) | 어려움 (항상 2+ AU) |
| 발사 창 | 4.71년 주기 (편심 궤도) | **상시** (원궤도) |
| 전이 시간 | 2~3년 | 1.0~1.5년 + NEP 감속 |
| 조성 리스크 | 높음 (미확인, 단일 소스) | **분산** (다수 천체) |
| 초기 투자 | **낮음** (수송선만) | 높음 (Mass Driver 건설) |
| 확장성 | 제한적 | **무제한** |

### 8.3. 솔직한 비교: 벨트는 1986 DA보다 모든 면에서 어렵다

| | 1986 DA | 소행성 벨트 | 벨트가 불리한 이유 |
|--|:---:|:---:|---|
| 총 Δv | 9.3 km/s | 11.0 km/s | 더 멀다 |
| 도착 감속 | **1.1 km/s** | **6.2 km/s** | 5.6배 — 가장 큰 격차 |
| 최소 추진제 | 14% (NTP) | 20% (Mars+NEP) | 인프라 없는 최선도 열세 |
| 인프라 요구 | NTP 수송선 몇 기 | NEP 터그 함대 or MD | 규모 차이 |
| 채광선 투입 | 쉬움 (0.21 AU) | 어려움 (2+ AU) | 항상 멀다 |

**벨트의 유일한 이점: 자원이 사실상 무한하다.** 이 한 가지가 나머지 모든 불리함을 상쇄할 만큼 크다.

### 8.4. 왜 그래도 성립하는가

**1) 추진제 20%는 "현지 조달 공짜"다**

벨트에서 추진제 20%라는 건 지구에서 20%를 실어 간다는 뜻이 아니다. C-type 소행성(벨트의 75%)에서 물을 퍼서 전기분해하면 H₂가 나온다. M-type 옆에 C-type이 널려 있다. **추진제 비용 = 채굴 에너지 비용**이지 수송 비용이 아니다.

**2) 어려움의 본질이 "물리적 불가능"이 아니라 "인프라 규모"다**

```
1986 DA:  수송선 몇 기 → 즉시 시작 가능
벨트:     NEP 터그 수백 기 or Mass Driver → 자기복제가 전제
```

자기복제가 확립된 후에는 NEP 터그든 Mass Driver든 SEL5 공장에서 찍어낸다. 문제는 "할 수 있냐"가 아니라 **"언제 할 수 있냐"**다.

**3) 1986 DA가 시간을 벌어준다**

1986 DA의 자원량은 ~수십억 톤. 연간 20만 톤 소비 시 **만 년 이상**. 벨트는 급하지 않다. 1986 DA를 캐면서 천천히 벨트 인프라를 구축하면 된다. 1986 DA의 조성이 기대 이하(시나리오 C, 확률 10~15%)인 경우에만 벨트 전환이 시급해진다.

### 8.5. 단계별 전환 논리

1986 DA와 소행성 벨트는 경쟁 관계가 아니라 **순차적 관계**다. 각 단계의 인프라 수준에 맞는 아키텍처가 있다:

```
Phase 1 (2038~):  1986 DA 직접 수송 → EML5 Bootstrap
                  ↳ NTP 수송선 몇 기. 도착 Δv 1.1 km/s. 인프라 불필요.
                  ↳ 추진제 14%. 가장 쉬운 경로.

Phase 2 (2050~):  벨트 진출 — 화성 경유 풀 NEP
                  ↳ Mass Driver 없이, NEP 터그만으로 시작.
                  ↳ 화성 스윙바이로 도착 Δv₂를 ~2 km/s로 감소.
                  ↳ 추진제 20%. C-type에서 현지 조달.
                  ↳ 고정 인프라 제로. 즉시 가동 가능.

Phase 3 (2060~):  벨트 확장 — Mass Driver 건설
                  ↳ 자기복제로 0.4~1.2 km 트랙 + 에너지 저장 시설 건설.
                  ↳ 출발 Δv를 무추진제로 전환.
                  ↳ 추진제 10% 미만. 터그 함대 1/3로 축소.

Phase 4 (2070~):  전이궤도항법 성숙
                  ↳ 다수 소행성에 MD 설치, 부표 네트워크 완비.
                  ↳ 화성 스윙바이 정밀 조준 실증 완료.
                  ↳ 벨트 전역에서 무한 자원 공급.
```

**핵심: Mass Driver는 Phase 3에서 등장한다.** Phase 2에서는 Mass Driver 없이 화성 경유 NEP만으로 벨트 수송이 가능하다. 진입 장벽을 낮추고, 인프라는 점진적으로 구축한다.
