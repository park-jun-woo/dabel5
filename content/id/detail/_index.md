Setiap klaim di DABEL5 didukung oleh perhitungan. Bagian ini mengumpulkan data desain kuantitatif: estimasi kinerja modul, neraca panas, parameter mekanika orbit, dan kebutuhan sumber daya.

Angka-angka asli di balik data yang disebutkan di bagian ikhtisar dapat ditemukan di sini.
