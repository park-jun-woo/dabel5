## 前の記事で見落としていたこと

[前の記事](/ja/why/why-iron-nickel-battery/)で鉄ニッケル電池がリチウムイオンに勝つ理由を示した。小惑星にリチウムはなく、真空中で火災を消せず、鉄ニッケルは30~50年持ち、過充電すれば水素を生む。

すべて正しい。**しかし一つ見落としていた。**

ダイソンモジュールは太陽熱発電所だ。ミラーが光を集め、熱でタービンを回す。eclipse（食）に備えてエネルギーを貯蔵する必要があるとき、現行設計はこうなっている：

```
太陽熱 (1,600°C) → タービン → 電気 (370 MW)
                         → 余剰電力 (~50 MW)
                              → バッテリー (化学エネルギー)    ← 変換2回
                              → eclipse時 → 電気に復帰        ← 変換3回
```

熱 → 電気 → 化学 → 電気。**変換3回。** 各段階で20~30%の損失。

熱を直接貯蔵すれば？

```
太陽熱 (1,600°C) → 一部を蓄熱槽に直接貯蔵               ← 変換0回
                → eclipse時 → 蓄熱槽 → タービン → 電気    ← 変換1回
```

**変換1回。** 効率の差は圧倒的だ。

太陽熱発電所の余剰エネルギーを電気に変え、化学に変え、また電気に戻すのは、水を水蒸気にし、水素と酸素に分解してからまた合成して水に戻すようなものだ。**できるが、なぜ？**

熱貯蔵が答えだ。では地球ではなぜやらないのか？

## 地球でできない理由、宇宙でできる理由

地球で溶融金属に熱を貯蔵するのは学術研究のテーマであり、産業の現実ではない。理由がある：

| 問題 | 地球 | 宇宙（無重力真空） |
|------|------|-------------------|
| 容器 | 数千トンの溶融物の重量を支えなければならない → 巨大で高価 | **自重なし** — 薄壁、あるいは非接触 |
| 断熱 | 対流+伝導+輻射をすべて遮断しなければならない | **輻射のみ遮断** — MLI数十層で完了 |
| 熱損失 | 高い — 空気対流が主因 | **極めて低い** — 真空で対流ゼロ |
| 腐食 | 1,500°C溶融物が壁面を侵食 | **電磁浮揚で非接触** → 腐食ゼロ |
| 安全 | 漏出時に大事故 | 真空で火災なし、漏出が広がる媒質なし |

**地球の弱点が宇宙ではすべて消えるか逆転する。** 前の記事で繰り返されたパターン — [タービン vs PV](/ja/why/why-turbines/)、鉄ニッケル vs リチウムイオン — とまったく同じ構造だ。

## 電磁浮揚蓄熱

溶融Fe-Niは1,500°Cでも電気伝導体だ（ニッケルのキュリー点以上で磁性は失うが、導電性は維持）。交流電磁場をかけると渦電流（eddy current）が誘導され、渦電流と磁場の反発力で**非接触浮揚**が可能になる。

地球でも実験室で使われている技術だ。[EML（Electromagnetic Levitation）](https://en.wikipedia.org/wiki/Electromagnetic_levitation)溶解と呼ばれる。数gから数kgの金属試料を空中に浮かせて溶かす。地球でもっと大きくできない理由はただ一つ — **重力**。重力に打ち勝つには磁場を強くしなければならず、強い磁場はエネルギーを食う。数kgが限界。

無重力では？**打ち勝つべき重力がない。** 位置安定化に必要な最小限の磁場だけでよい。数トンでも、数百トンでも、数万トンでも。

```
[蓄熱ユニット断面]

        +--- MLI反射壁（多層反射断熱）---+
        |                                |
        |    +-- 電磁コイル（冷却）--+    |
        |    |                      |    |
        |    |   @@@@@@@@@@@@@@     |    |
        |    |   @ 溶融Fe-Ni塊   @   |    |
        |    |   @ (1,200~1,500°C) @   |    |
        |    |   @@@@@@@@@@@@@@     |    |
        |    |                      |    |
        |    +----------------------+    |
        |                                |
        +--------------------------------+
```

無重力で溶融金属は表面張力により**自然に球形**になる。球形は体積あたりの表面積が最小 — 輻射熱損失が最小。MLI反射壁が輻射熱を閉じ込め、電磁場が位置を保ち、壁面とは非接触なので腐食がゼロ。

**小惑星から採掘したFe-Niを溶かして浮かべておけば蓄熱槽だ。**

## 充電と放電

```
[充電 — 通常時]
太陽熱集光 → 輻射シャッター開放 → 金属塊加熱 → 1,200°C → 1,500°C

[放電 — eclipse時]
輻射シャッター開放 → 金属塊の輻射熱が熱交換器を加熱 → 作動流体 → タービン
1,500°C → 1,200°C (ΔT=300°Cを活用)
```

充電：ミラーが集めた太陽熱の一部を蓄熱槽方向に送ればよい。シャッターを開ければ光が金属塊を加熱する。

放電：eclipseが来たらシャッターを開け、金属塊の輻射熱を熱交換器が受ける。熱交換器が作動流体を加熱してタービンを回す。既存のタービンをそのまま使う — 通常時はミラーが熱源、eclipse時は蓄熱槽が熱源。**タービンにとっては熱源が変わるだけで、あとは同じ。**

熱交換の媒介は輻射だ。非接触溶融体にパイプを差し込むことはできないので、輻射シャッターを介した熱伝達が基本メカニズムとなる。1,500°C溶融金属の輻射エネルギーはステファン-ボルツマン法則によりT⁴に比例 — 十分に強力だ。

## エネルギー密度：比熱 + 潜熱

Fe-Ni合金の比熱：~0.5 kJ/(kg·K) = ~0.14 Wh/(kg·K)。温度変化(ΔT)に比例する**顕熱（sensible heat）**のみ計算すると：

| 温度範囲 (ΔT) | 顕熱 | 備考 |
|----------------|------|------|
| 300°C (1,200→1,500°C) | ~42 Wh/kg | 保守的 |
| 500°C (1,000→1,500°C) | ~70 Wh/kg | 中間 |
| 1,000°C (500→1,500°C) | ~140 Wh/kg | 積極的 |

しかしここで終わりではない。

### 潜熱ボーナス

Fe-Ni合金の融点は~1,430~1,450°C。運用範囲1,000~1,500°Cはこの融点を**貫通する。** 充電時に金属が溶け、放電時に固まる — 相変化（phase change）。

物質が溶けるとき温度は上がらないが膨大な熱を吸収する。これが**融解潜熱（latent heat of fusion）。**

```
鉄(Fe) 融解潜熱: ~270 kJ/kg ≈ 75 Wh/kg
Fe-Ni 合金: 類似範囲
```

顕熱と潜熱を合算すると：

| 温度範囲 | 顕熱 | 潜熱 | **合計** |
|----------|------|------|---------|
| 300°C (1,200→1,500°C) | ~42 | ~75 | **~117 Wh/kg** |
| 500°C (1,000→1,500°C) | ~70 | ~75 | **~145 Wh/kg** |
| 1,000°C (500→1,500°C) | ~140 | ~75 | **~215 Wh/kg** |

**潜熱一つでエネルギー密度が2倍。** 鉄の塊が溶けて固まるだけでリチウムイオン電池（150~270 Wh/kg）の下限と重なる。

### ESS比較（潜熱含む）

| 方式 | エネルギー密度 | サイクル寿命 | 素材調達 |
|------|-----------|-----------|----------|
| リチウムイオン | 150~270 Wh/kg | 3,000~10,000回 | 不可（小惑星にLiなし） |
| 鉄ニッケル電池 | 30~50 Wh/kg | 事実上無限 | 小惑星Fe-Ni |
| **溶融Fe-Ni蓄熱** | **117~215 Wh/kg** | **事実上無限** | **小惑星Fe-Ni** |

リチウムイオンとエネルギー密度が同等でありながら、サイクル寿命は無限、素材は小惑星にいくらでもある。そして熱→電気変換が1回だけなのでシステム効率まで圧倒的。

サイクル寿命が無限の理由：金属の塊を熱くして冷ますだけだ。化学反応がない。電極がない。電解質がない。劣化させるもの自体がない。

## 規模：なぜ巨大な球一つではなく小さなユニット60個なのか

eclipse最大12時間、タービン出力370 MW。全部を蓄熱でカバーする必要はない — H₂燃料電池とバッテリーが分担する。

### ハイブリッド算定

```
eclipse 12時間中:
  蓄熱槽: 6時間分
  H₂ 燃料電池: 4時間分（バソライザー年間蓄積分）
  鉄ニッケル電池: 2時間分（瞬時負荷追従 + バックアップ）
```

蓄熱槽6時間分（潜熱含む）：

```
370 MW ÷ 0.30 (タービン効率) = ~1,233 MW(th) × 6h = ~7,400 MWh(th)

ΔT=500°C + 潜熱基準 (145 Wh/kg):
  必要質量 = 7,400,000 kWh ÷ 0.145 kWh/kg = ~51,000 トン

（潜熱なしの場合105,000トン → 潜熱ボーナスで質量半減）
```

51,000トンを一つの球に入れると半径~12 m。直感的にはシンプルだ。**しかしこれはダメだ。** 三つの工学的理由がある。

### 理由1：放電時に表面積が足りない

eclipse時、蓄熱体は**輻射のみ**で熱交換器に熱を伝える。輻射出力は表面積に比例する（P = ε σ A T⁴）。

球形は体積あたり表面積が最小の形状だ。熱を**保管**するときは最適だが、熱を**素早く放出**するときはボトルネックになる。

```
必要熱出力: ~1,233 MW(th)

1,500°C(1,773K) 輻射出力 (ε=0.5):
  P/A = ε × σ × T⁴ = 0.5 × 5.67e-8 × 1,773⁴ ≈ 280 kW/m²

必要表面積: 1,233,000 kW ÷ 280 kW/m² ≈ 4,400 m²

半径12m単一球の表面積: 4π(12)² ≈ 1,810 m² → 不足（必要量の41%）
```

**単一の球では物理的に必要な熱を放出できない。** 表面積が半分にも満たない。

半径3 mユニット~58個に分割すると：

```
ユニット1個の表面積: 4π(3)² ≈ 113 m²
58個の総表面積: 113 × 58 ≈ 6,560 m² → 必要量の149%（余裕あり）
ユニット1個の質量: (4/3)π(3)³ × 7,800 ≈ 880 トン
```

**貯蔵時は各ユニットが球形を維持して損失を最小化し、放電時は複数ユニットの総表面積で十分な熱出力を確保する。** 球形の欠点をユニット数で解決。

### 理由2：スロッシング — 10万トンの溶岩レッキングボール

51,000トンの液体金属が単一球で浮いているとき、モジュールが姿勢制御のためにわずかでも回転や振動すると内部に**巨大な波（スロッシング）**が発生する。磁気流体力学（MHD）不安定性が重なるとこの溶岩の塊が揺れ動いて電磁場の拘束を突破する危険がある。

半径3 m、880トンのユニットなら？流動エネルギーはユニットサイズの三乗に比例するため**個々のユニットのスロッシングエネルギーは単一球の1/10,000以下**に減少する。拘束離脱リスクは事実上ゼロになる。

### 理由3：相変化時の体積膨張

1,200°C（固体）と1,500°C（液体）を行き来するとFe-Niは膨張と収縮を繰り返す。半径12 mの球が表面から冷えると固体の殻が形成され、内部液体が収縮しながら**殻が割れて破片が真空に飛び出す**危険がある。小型ユニットは内外の温度勾配が均一に管理され、この問題が解消される。

### 設計結論

```
蓄熱ユニット仕様:
  形状: 球形（表面張力による自然形成）
  半径: ~3 m
  質量: ~880 トン/ユニット
  ユニット数: ~58個（モジュールあたり）
  総質量: ~51,000 トン
  配置: ミラー後方構造物内に分散配置（カウンターウェイト兼用）

放電性能:
  総表面積: ~6,560 m²（必要量4,400 m²の149%）
  1,233 MW(th) 出力余裕確保
```

51,000トンは別途調達するのではない。小惑星から精錬したFe-Niを**溶かしたまま固めずに置いておけば蓄熱ユニット**だ。モジュール構造物に分散配置すれば**カウンターウェイト**も兼ねる。

## 3層ESS：役割分離

バッテリーがバルクESSを担う必要がなくなる。各層に最適な技術を配置する：

```
第1層 — バルク（時間単位）
  └→ 溶融金属蓄熱槽
       充電: 太陽熱直接
       放電: 蓄熱 → タービン → 電気
       役割: eclipse対応、変換損失最小

第2層 — バッファ（秒~分単位）
  └→ 鉄ニッケル電池
       充電: 余剰電力
       放電: 電気化学（ms応答）
       役割: 瞬時負荷追従、起動電力

第3層 — 緊急 + 化学生産
  └→ H₂/O₂（バソライザー産出物）
       燃料電池緊急発電
       推進剤・還元剤・呼吸用酸素
       eclipse長期化時の二次バックアップ
```

### この構造がもたらすもの

**バッテリーバンクが大幅に縮小する。** 以前の設計で12時間eclipseをバッテリーだけでカバーすると111,000 m³が必要だった。蓄熱槽がバルクを担えばバッテリーは2時間分 — 数千m³に縮小。

**バソライザーの役割が明確になる。** 前の記事でバソライザー（過充電時の水電解）をESS機能と化学生産を兼ねるものとして説明した。蓄熱槽がバルクESSを担えばバソライザーは**化学プラント**に位置づけられる — 水素推進剤、酸素、還元剤の生産が本業で、緊急発電は副業。

**素材が同一だ。** 蓄熱槽 = 溶融Fe-Ni。バッテリー = Fe-Ni電極。バソライザー = 同じバッテリーに過充電。3層すべてが小惑星Fe-Niから出る。自己複製ループに新しい原料は追加されない。

## 前の記事（鉄ニッケル電池）との関係

前の記事の核心論旨は**すべて有効だ：**

- 小惑星にリチウムがない → そのまま
- 鉄ニッケル電池の30~50年寿命 → そのまま
- 真空での火災リスク → そのまま
- バソライザーのH₂/O₂生産 → そのまま
- 現地製造可能 → そのまま

**補完する部分：** 鉄ニッケル電池がバルクESS（eclipse 12時間対応）まで一人で担うように読めた可能性がある。実際にはバルクエネルギー貯蔵では熱貯蔵が圧倒的に有利で、バッテリーは瞬時応答という自分の領域で輝く。

**各自が得意なことをすればいい。** 溶鉱炉は時間単位の熱貯蔵。バッテリーはミリ秒単位の電力応答。燃料電池は緊急対応+化学生産。一つですべてを賄う必要はない。

## 一行まとめ

ダイソンモジュールは太陽熱発電所なのに、熱を電気に変え化学に変えまた電気に戻すのは三重変換損失だ。小惑星Fe-Niを溶かして無重力に浮かべれば変換0回充電、1回放電の蓄熱槽になる。相変化潜熱を合算すればエネルギー密度~145 Wh/kg — リチウムイオンと同等。半径3 mユニット58個を分散配置して放電時の表面積ボトルネック、スロッシング、相変化膨張を解決する。素材はすべて同じ小惑星Fe-Ni。
