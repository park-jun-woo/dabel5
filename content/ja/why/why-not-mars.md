## 「ダイソンスウォームがあったら、人はどこに住むのか？」

気候制御、小惑星採掘、鏡が鏡を作る設計を示してきた。自然と続く問いがある：**では人はどこに住むのか？**

ほとんどの人は「火星」と答える。数十年にわたるSFが固めてきたイメージ。SpaceXがロケットを作っている。火星は人類の第二の故郷——これはほぼ常識として定着している。

しかし工学的に分析すると、火星は人類の居住地として**致命的な欠陥**がある。そしてその欠陥は技術では解決できない。

---

## 0.38G——誰も知らない数字

火星の表面重力は0.38G。地球の38%。「重力があるなら問題ないだろう」と考えがちだが、問題は**どれだけあるか**だ。

人間の骨、筋肉、心血管系、内耳（前庭器官）は1Gに最適化されて進化した。ISSで0Gに6か月滞在した宇宙飛行士は、骨密度低下、筋萎縮、視力低下を経験する。0Gが有害であることはわかっている。

**0.38Gはどうか？**

正直な答え：**誰にもわからない。**

- NASAとESAの宇宙医学データには0G（ISS）と1G（地球）の2点しかない
- 0.38Gで哺乳類の妊娠、胎児の骨格形成、内耳発達、心血管形成が正常に進行するか、**一度も実験されたことがない**
- 0.38Gで生まれ育った子供の骨と筋肉が1Gに耐えられるかわからない
- 耐えられなければ、火星で生まれた人間は**地球に来ることができない**

これは技術の問題ではない。火星の質量は変えられない。0.38Gは火星という惑星の**物理定数**だ。

### オニールシリンダー：重力を選ぶ

回転する円筒形構造物——オニールシリンダー（O'Neill cylinder）。回転半径と角速度を調整すれば、遠心力で内部に**正確に望む重力**を生み出せる。

| | 火星 | オニールシリンダー |
|---|---|---|
| 重力 | 0.38G（固定） | **0.5〜1.5G（選択可能）** |
| 1G保証 | 不可能 | 回転速度の調整で保証 |
| 世代間の健康 | **未知** | 地球と同一条件 |
| 地球帰還の可能性 | 不確実 | 保証 |

火星の利点は「重力がある」ことではなく、**「重力が中途半端だ」**ことが問題だ。オニールシリンダーは重力を**設計変数**にする。

## エネルギー：ダイソンモジュールのすぐ隣

火星に都市を建てたら、エネルギーはどこから調達するのか？

| | 火星 | L5（ダイソンモジュール隣接） |
|---|---|---|
| 太陽光フラックス | 589 W/m²（地球の43%） | 1,361 W/m²（100%） |
| 利用可能時間 | 昼間のみ（砂嵐時は数か月遮断） | **365日24時間無中断** |
| エネルギーインフラ | 発電所を**ゼロから**建設 | ダイソンモジュールから**直接供給** |

L5では、居住モジュールはダイソンモジュールクラスターの一部だ：

```
ダイソンモジュール（370 MW電力）
    |-> 電力 -> 生命維持、農業照明、生活電力
    |-> 熱カスケード
         |-> 100〜200°C -> 居住区の暖房・給湯（無料）
         |-> 30〜60°C -> データセンターの環境熱
```

モジュール1基あたり370 MW電力、3,000人居住。**1人あたり約120 kW。**地球先進国の平均消費電力の10倍以上。居住区のエネルギーはダイソンモジュールの一機能に過ぎず、別途の発電所は不要だ。

火星で同レベルのエネルギーを供給するには？砂嵐の中で巨大な太陽光アレイを建設しなければならない。しかも夜になれば停止する。

## 産業：工場はすでにある

火星に都市を建てるには、**まず工場を建てなければならない。**建設資材、生活用品、電子部品、医療機器——すべてを地球から送るか、火星でゼロから産業基盤を構築するかだ。

火星の産業環境：
- 大気：CO₂ 95%、気圧は地球の0.6% -> **宇宙服なしでは屋外活動不可**
- 水：極地の氷冠に存在するが、採取・精製インフラが必要
- 原材料：存在するが、重力井戸の底での採掘 -> 軌道まで持ち上げるコストが膨大

L5の産業環境：
- ダイソンモジュールクラスター**自体が産業団地**
- 精錬モジュール -> 構造材、パイプ、バッテリー
- ファブモジュール -> 電子部品、センサー、AIチップ
- 構造材モジュール -> 居住区拡張部材
- 生活必需品（衣類、工具、医療機器）を**現地で製造可能**

火星に都市を建てるには工場を先に建てる必要がある。L5では**工場はすでに稼働している。**居住モジュールの追加は、既存の生産ラインに部屋を一つ加えるようなものだ。

## 通信：火星の孤立

ここからは技術的不可能ではなく、**生活の質**の問題だ。しかし文明の居住地を論じる際に生活の質を除外することはできない。

| | 火星 | L5（SEL5） |
|---|---|---|
| 地球までの通信遅延 | **4〜24分**（片道、位置により変動） | **約8分20秒**（片道、一定） |
| 通信途絶 | 合（conjunction）時に**約2週間**完全途絶 | **なし** |
| リアルタイム会話 | 不可能（往復8〜48分） | 不可能（往復約17分） |

どちらもリアルタイム通話はできない。しかし火星には地球と**完全に**断絶される2週間がある。L5にはない。

そして決定的な違い：L5の居住ハブはSEL5ではなく**EML4/5**に配置できる。

```
EML4/5居住ハブ
  |-> 地球までの通信：往復約2.6秒（リアルタイムビデオ通話可能！）
  |-> SEL5産業団地との交代勤務シャトル
  |-> 月まで2〜3日（物流・観光）
```

EML4/5から地球まで往復2.6秒なら、**電話ができる。**火星とは比較にすらならない。

## 帰還：火星は事実上片道

火星から地球に戻るには：

- 火星表面 -> 火星軌道：Δv ~3.8 km/s
- 火星軌道 -> 地球軌道：Δv ~2 km/s以上
- **合計Δv ~5.7 km/s + 飛行時間9か月**
- 打ち上げウィンドウ：26か月に1回

経済的にも心理的にも、火星移住は**片道**だ。「いつでも帰れる」は理論的には正しいが、現実にはほぼ不可能だ。

L5 -> 地球：
- EML居住ハブから地球軌道まで：Δv < ~1 km/s
- 物流インフラ（タグボート、シャトル）はすでに存在
- **定期シャトル運行が可能**

火星に行くということは「移民」だ。L5に行くということは「出張」または「引っ越し」だ。帰って来られるかどうかは、居住地選択における根本的な違いだ。

## テラフォーミングという幻想

「火星をテラフォーミングすればいいのでは？」

テラフォーミングの現実：

- **時間**：数百〜数千年。楽観的な見積もりでも少なくとも数世紀
- **大気の維持**：火星には地球型の磁場がない。太陽風が大気を継続的に剥ぎ取る。大気を作っても**維持する方法が不確実**
- **重力**：大気を作っても0.38Gは変わらない。大気が地球並みに厚くなっても重力問題はそのまま

オニールシリンダーは：
- 建設**即座に**1 atm、1G
- 内部に山、川、空を作れる（シリンダー内部の景観設計）
- 望む生態系を即座に実現
- テラフォーミングを待つ必要がない

テラフォーミングの完成を数百年待つ間に、オニールシリンダーでは**すでに人が暮らしている。**

## 「それでも惑星に住みたい」

工学的論証だけでは説得しきれない部分がある。足元に惑星がある心理的安心感。窓の外に見える空と地平線。それは認める。

しかしいくつか指摘しておく：

**「火星は資源が豊富だ」**——資源のアクセス性では小惑星の方が優れている。火星は重力井戸の底にあり、採掘した資源を軌道に上げるのに膨大なエネルギーがかかる。小惑星は重力がほぼゼロで、持っていくだけだ。

**「オニールシリンダーは小さすぎる」**——モジュール数を増やせばいい。モジュール10,000基で人口3,000万人。自己複製は指数関数的に拡大するため、テラフォーミングが完了するはるか前にL5の人口が火星を超える。

**「心理的に惑星が必要だ」**——オニールシリンダーの内部に山と川と雲を作れる。半径数kmのシリンダーなら空もある。窓の外の景色が火星の赤い砂漠かオニールシリンダーの設計された自然か——好みの問題かもしれないが、工学的優劣は明確だ。

**「SpaceXがすでに投資している」**——火星探査と火星居住は別の問題だ。探査基地は南極の昭和基地のようなものだ。そこで文明を築こうとは誰も言わない。探査基地 ≠ 文明の居住地。

## 一行まとめ

火星の0.38Gで子供が正常に成長できるかどうか、誰にもわからない。オニールシリンダーなら1Gが保証される。エネルギーはダイソンモジュールから直接供給され、工場はすでに稼働し、地球に電話もできる。火星はロマンチックで、L5はエンジニアリングだ。
