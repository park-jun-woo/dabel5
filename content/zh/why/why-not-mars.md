## "有了戴森群，人住哪里？"

我们已经展示了气候控制、小行星采矿、以及镜子制造镜子的设计。随之而来一个自然的问题：**人到底住在哪里？**

大多数人的答案是"火星"。几十年来的科幻作品巩固了这一形象。SpaceX正在造火箭。火星是人类的第二故乡——这几乎已成为常识。

但从工程角度分析，火星作为人类居住地存在**致命缺陷**。而且这些缺陷无法靠技术解决。

---

## 0.38G——没有人知道的数字

火星表面重力为0.38G，是地球的38%。"有重力不就行了吗？"很容易这样想，但问题在于**有多少**。

人类的骨骼、肌肉、心血管系统和内耳（前庭器官）是在1G环境下进化优化的。在ISS的0G环境中停留六个月的宇航员会出现骨密度下降、肌肉萎缩和视力退化。0G有害，这我们知道。

**0.38G呢？**

诚实的答案：**没有人知道。**

- NASA和ESA的太空医学数据只有两个数据点：0G（ISS）和1G（地球）
- 哺乳动物在0.38G下的妊娠、胎儿骨骼形成、内耳发育、心血管形成能否正常进行，**从未做过哪怕一次实验**
- 我们不知道在0.38G出生和成长的孩子的骨骼和肌肉能否承受1G
- 如果不能，在火星出生的人**永远无法来到地球**

这不是技术问题。你无法改变火星的质量。0.38G是火星这颗行星的**物理常数**。

### 奥尼尔圆柱体：选择你想要的重力

旋转的圆柱形结构——奥尼尔圆柱体（O'Neill cylinder）。通过调节旋转半径和角速度，可以用离心力在内部产生**精确所需的重力**。

| | 火星 | 奥尼尔圆柱体 |
|---|---|---|
| 重力 | 0.38G（固定） | **0.5~1.5G（可选）** |
| 1G保证 | 不可能 | 通过调节转速保证 |
| 代际健康 | **未知** | 与地球相同条件 |
| 返回地球可能性 | 不确定 | 保证 |

火星的优势不是"有重力"——问题在于**"重力不够明确"**。奥尼尔圆柱体将重力变成了**设计变量**。

## 能源：紧邻戴森模块

在火星建城市，能源从哪来？

| | 火星 | L5（戴森模块旁） |
|---|---|---|
| 太阳辐射通量 | 589 W/m²（地球的43%） | 1,361 W/m²（100%） |
| 可用时间 | 仅白天（沙尘暴期间数月受阻） | **365天24小时不间断** |
| 能源基础设施 | **从零开始**建发电站 | 从戴森模块**直接供给** |

在L5，居住模块是戴森模块集群的一部分：

```
戴森模块（370 MW电力）
    |-> 电力 -> 生命维持、农业照明、生活用电
    |-> 热级联
         |-> 100~200°C -> 居住区供暖·热水（免费）
         |-> 30~60°C -> 数据中心环境热
```

一个模块产生370 MW电力，供3,000人居住。**人均约120 kW。**这是地球发达国家平均用电量的10倍以上。居住区的能源只是戴森模块的一个附属功能——不需要单独的发电站。

要在火星提供同等水平的能源？你得在沙尘暴中建造巨大的太阳能阵列。而且一到夜晚就停转。

## 工业：工厂已经在了

要在火星建城市，**首先得建工厂。**建筑材料、生活用品、电子元器件、医疗设备——全部要从地球运来，或者在火星从零构建工业基础。

火星的工业环境：
- 大气：95% CO₂，气压为地球的0.6% -> **不穿宇航服无法进行户外活动**
- 水：存在于极地冰盖，但需要开采和净化基础设施
- 原材料：存在，但在引力井底部采矿 -> 将资源送上轨道的成本巨大

L5的工业环境：
- 戴森模块集群**本身就是工业园区**
- 冶炼模块 -> 结构材料、管道、电池
- 晶圆厂模块 -> 电子元器件、传感器、AI芯片
- 结构模块 -> 居住区扩建构件
- 日用品（衣物、工具、医疗设备）**本地制造**

要在火星建城市，需要先建工厂。在L5，**工厂已经在运转了。**添加居住模块更像是在现有生产线上加一个房间。

## 通信：火星的孤立

从这里开始，不再是技术上的不可能——而是**生活质量**问题。但讨论文明的居住地时，不能忽略生活质量。

| | 火星 | L5（SEL5） |
|---|---|---|
| 到地球的通信延迟 | **4~24分钟**（单程，因位置而异） | **约8分20秒**（单程，恒定） |
| 通信中断 | 合相期间**约2周**完全中断 | **无** |
| 实时对话 | 不可能（往返8~48分钟） | 不可能（往返约17分钟） |

两者都无法实时通话。但火星有长达两周与地球**完全**断联的时期。L5没有。

而决定性的区别在于：L5的居住中心可以不放在SEL5，而放在**EML4/5**。

```
EML4/5居住中心
  |-> 与地球通信：往返约2.6秒（可实时视频通话！）
  |-> 与SEL5工业园区轮班通勤班车
  |-> 到月球2~3天（物流·旅游）
```

从EML4/5到地球往返2.6秒，**可以打电话。**与火星根本没有可比性。

## 返回：火星实质上是单程

从火星返回地球需要：

- 火星表面 -> 火星轨道：Δv ~3.8 km/s
- 火星轨道 -> 地球轨道：Δv ~2 km/s以上
- **总Δv ~5.7 km/s + 9个月飞行时间**
- 发射窗口：每26个月一次

从经济和心理角度，火星移民实质上是**单程**的。"随时可以回来"理论上成立，但实际上几乎不可能。

L5 -> 地球：
- 从EML居住中心到地球轨道：Δv < ~1 km/s
- 物流基础设施（拖船、班车）已经存在
- **可运营定期班车**

去火星意味着"移民"。去L5意味着"出差"或"搬家"。能否回来，是选择居住地时的根本性差异。

## 地球改造的幻想

"改造火星不就行了？"

地球改造的现实：

- **时间**：数百至数千年。即使乐观估计也至少需要几个世纪
- **大气维持**：火星没有地球型磁场。太阳风持续剥离大气层。即使创造了大气，**能否维持尚不确定**
- **重力**：即使有了大气，0.38G不会改变。即使大气变得和地球一样厚，重力问题依然存在

奥尼尔圆柱体：
- 建成**即刻**拥有1 atm、1G
- 内部可建造山川、河流和天空（圆柱体内部景观设计）
- 任何所需生态系统即刻实现
- 无需等待地球改造

在等待数百年地球改造完成的过程中，奥尼尔圆柱体里的人**已经在生活了。**

## "但我还是想住在行星上"

仅靠工程论证无法说服所有人。脚下有行星的心理安全感。窗外可见的天空和地平线。这一点我们承认。

但有几点需要指出：

**"火星资源丰富"**——就资源可及性而言，小行星更优。火星位于引力井底部，将开采的资源提升到轨道需要巨大能量。小行星几乎没有重力——直接取用即可。

**"奥尼尔圆柱体太小了"**——增加模块数量即可。10,000个模块意味着3,000万人口。自我复制呈指数级扩展，因此在地球改造完成之前，L5人口早已超过火星。

**"心理上需要行星"**——奥尼尔圆柱体内部可以建造山川河流和云彩。半径数公里的圆柱体甚至有天空。窗外是火星的红色荒漠还是奥尼尔圆柱体的设计自然——这可能是品味问题，但工程优劣是明确的。

**"SpaceX已经在投资了"**——火星探索和火星定居是不同的问题。探索基地就像南极的世宗基地。没有人建议在那里建立文明。探索基地 ≠ 文明的居住地。

## 一句话总结

没有人知道孩子能否在火星0.38G环境下正常发育。奥尼尔圆柱体保证1G。能源直接来自戴森模块，工厂已经在运转，甚至可以给地球打电话。火星是浪漫的，L5是工程的。
