# post012 — 왜 용융금속 축열이 우주 ESS의 1차 수단인가

> 트리거: post011(철-니켈 전지)에서 "리튬 vs 철-니켈" 프레임을 세웠으나, 진짜 경쟁자는 열 저장이었다. 다이슨 모듈은 태양열 발전소인데, 잉여 열을 전기→화학→전기로 이중 변환하는 건 비효율적. 용융 금속에 직접 열을 담으면?

## 핵심 논지

다이슨 모듈의 벌크 ESS(eclipse 대응)는 **용융금속 축열조**가 최적. 철-니켈 배터리는 보조(순간 부하 추종 + 바톨라이저)로 역할 분리.

---

## 1. 왜 열을 전기로 바꿔서 저장하나?

다이슨 모듈의 에너지 흐름:

```
태양열 (1,600°C) → 터빈 → 전기 (370 MW)
                         → 잉여 전력 (~50 MW)
                              → 배터리 (화학 에너지)     ← 변환 2회
                              → eclipse 시 → 전기로 복귀  ← 변환 3회
```

열 저장이면:

```
태양열 (1,600°C) → 일부를 축열조에 직접 저장          ← 변환 0회
                → eclipse 시 → 축열조 → 터빈 → 전기   ← 변환 1회
```

**변환 횟수: 3회 → 1회.** 매 변환마다 20~30% 손실. 열 저장이 시스템 효율에서 압도적.

---

## 2. 왜 지구에서는 안 하나?

지구에서 용융금속 축열이 비주류인 이유:

| 문제 | 지구 | 우주 (무중력 진공) |
|------|------|-------------------|
| 용기 구조 | 수천 톤 용융물 중량 지지 → 거대하고 비쌈 | **자중 없음** — 얇은 벽 or 비접촉 |
| 단열 | 대류+전도+복사 모두 차단 필요 | **복사만 차단** — MLI 수 겹이면 끝 |
| 열 손실 | 높음 (공기 대류가 주범) | **극히 낮음** — 진공에서 대류 제로 |
| 용기 부식 | 고온 용융물이 벽면 침식 | **전자기 부유로 비접촉 가능** |
| 안전 | 누출 시 대형 사고 | 진공이라 화재 없음, 누출 범위 제한 |

**지구의 약점이 우주에서는 전부 사라지거나 역전된다.**

## 3. 전자기 부유 축열 (Electromagnetic Levitation Thermal Storage)

### 원리

용융 Fe-Ni는 1,500°C 이상에서도 전기 전도체 (비자성이지만 도전성 유지). 교류 전자기장으로 와전류를 유도하면 비접촉 부유(levitation) 가능.

지구에서 이미 사용되는 기술:
- **EML (Electromagnetic Levitation) 용해**: 실험실에서 수 g~수 kg 금속 시료를 부유 용해
- 지구에서는 중력 극복에 대부분의 에너지가 소모 → 소량만 가능
- **무중력에서는 중력 극복 불필요** → 수 톤~수백 톤 규모로 스케일 가능

### 구조 개념

```
[축열 유닛 단면]

        ┌─── MLI 반사벽 (다층 반사 단열) ───┐
        │                                    │
        │    ┌── 전자기 코일 (냉각) ──┐      │
        │    │                        │      │
        │    │   ●●●●●●●●●●●●●●●    │      │
        │    │   ● 용융 Fe-Ni 덩어리 ●    │      │
        │    │   ● (1,200~1,500°C)  ●    │      │
        │    │   ●●●●●●●●●●●●●●●    │      │
        │    │                        │      │
        │    └────────────────────────┘      │
        │                                    │
        └────────────────────────────────────┘

- 용융 금속은 표면장력으로 구형 유지 (무중력)
- 구형 = 표면적/부피 비 최소 → 복사 손실 최소
- MLI 반사벽이 복사열을 가둠
- 전자기장이 위치 안정화 + 교반
- 벽면과 비접촉 → 부식 제로
```

### 충전/방전

```
[충전 — 평상시]
태양열 집광 → 열교환기 접근 → 금속 덩어리 가열 → 1,200→1,500°C

[방전 — eclipse 시]
금속 덩어리에 열교환기 접근 → 작동유체 가열 → 터빈 발전
1,500→1,200°C (ΔT=300°C 활용)
```

열교환기 접근/이탈은 기계적 이동 또는 복사 셔터로.

## 4. 에너지 밀도 (현열 + 잠열)

Fe-Ni 합금의 비열: ~0.5 kJ/(kg·K) = ~0.14 Wh/(kg·K)

**잠열 보너스:** Fe-Ni 합금 융점 ~1,430~1,450°C. 운용 범위 1,000~1,500°C가 융점을 관통하므로 매 사이클마다 상변화 잠열 추가. 철 융해 잠열 ~270 kJ/kg ≈ 75 Wh/kg.

| 온도 범위 (ΔT) | 현열 | 잠열 | **합계** |
|----------------|------|------|---------|
| 300°C (1,200→1,500°C) | ~42 | ~75 | **~117 Wh/kg** |
| 500°C (1,000→1,500°C) | ~70 | ~75 | **~145 Wh/kg** |
| 1,000°C (500→1,500°C) | ~140 | ~75 | **~215 Wh/kg** |

비교 (잠열 포함):

| 방식 | 에너지 밀도 | 사이클 수명 | 소재 조달 |
|------|-----------|-----------|----------|
| 리튬이온 | 150~270 Wh/kg | 3,000~10,000회 | 불가 |
| 철-니켈 배터리 | 30~50 Wh/kg | 사실상 무한 | 소행성 |
| **용융 Fe-Ni 축열** | **117~215 Wh/kg** | **사실상 무한** | **소행성** |

**리튬이온 하단과 동급 에너지 밀도, 무한 사이클, 소행성 소재.**

## 5. 규모 산정

### eclipse 대응 (벌크 ESS)

```
eclipse 최대 12시간, 터빈 370 MW

필요 열에너지 (터빈 효율 30% 고려):
  370 MW ÷ 0.30 = ~1,233 MW(th) × 12h = ~14,800 MWh(th)

ΔT=300°C 기준 (42 Wh/kg):
  필요 질량 = 14,800,000 kWh ÷ 0.042 kWh/kg = ~352,000 톤

ΔT=500°C 기준 (70 Wh/kg):
  필요 질량 = 14,800,000 kWh ÷ 0.070 kWh/kg = ~211,000 톤
```

모듈 차폐층이 수만 톤인 점 고려하면 수십만 톤은 크지만, eclipse가 연 3~12시간이고 이를 전부 축열로 커버할 필요 없음 (H₂ 연료전지 병행). 실제 필요량은 위의 1/4~1/2 수준.

### 현실적 하이브리드 산정 (잠열 포함)

```
eclipse 12시간 중:
  - 축열조: 6시간분 → ~51,000 톤 (ΔT=500°C + 잠열, 145 Wh/kg)
  - H₂ 연료전지: 4시간분 → 바톨라이저 연간 축적분
  - 철-니켈 배터리: 2시간분 → 순간 부하 + 백업

(이전 산정 105,000톤 → 잠열 보너스로 51,000톤으로 반감)
```

### 복수 유닛 설계 (단일 대형 구 불가)

세 가지 공학적 이유:
1. **방전 표면적 병목**: 반경 12m 단일 구 표면적 1,810 m² < 필요 4,400 m² (1,233 MW(th) at 280 kW/m²)
2. **슬로싱/MHD 불안정**: 5만 톤 액체금속 단일 구의 유동 불안정 → 구속 이탈 위험
3. **상변화 부피 팽창**: 겉부터 고체 껍질 형성 → 내부 수축 시 껍질 파괴

```
설계점:
  - 반경 ~3 m 구형 유닛 × ~58개
  - 유닛당 ~880 톤
  - 총 표면적 ~6,560 m² (필요량의 149%)
  - 구조물 내 분산 배치 (균형추 겸용)
```

## 6. 역할 분리: 축열 + 배터리 + 연료전지

```
[계층적 ESS 아키텍처]

1차 (벌크, 시간 단위)
  └→ 용융금속 축열조 — eclipse 대응, 변환 손실 최소
       충전: 태양열 직접
       방전: 축열 → 터빈 → 전기

2차 (버퍼, 초~분 단위)
  └→ 철-니켈 배터리 — 순간 부하 추종, 기동 전력
       충전: 잉여 전력
       방전: 전기화학 (ms 응답)

3차 (비상 + 생산)
  └→ H₂/O₂ (바톨라이저 산출물)
       연료전지 비상 발전
       추진제·환원제·호흡용 산소 생산
       eclipse 장기화 시 2차 백업
```

### 이 구조의 장점

- **벌크 ESS에서 배터리 제거** → 배터리 뱅크 대폭 축소 (111,000 m³ → 수천 m³)
- **축열조 소재 = 원광 그 자체** → 제련된 Fe-Ni를 그냥 녹인 채로 두면 축열조
- **바톨라이저는 ESS가 아닌 화학 공장으로 포지셔닝** → 목적이 명확해짐
- 3계층 모두 소행성 Fe-Ni에서 나옴 → 자기복제 루프 유지

## 7. 기존 글(post011)과의 관계

post011(왜 리튬이 아니라 철-니켈인가)의 핵심 논지는 유효:
- 리튬 부재, 화재 위험, 수명, 현지 제조 — 모두 그대로
- **수정 필요 사항:** 철-니켈 배터리가 벌크 ESS의 유일한 답인 것처럼 서술된 부분을 보완
- **추가 글 방향:** `/power/` 섹션에 "다이슨 모듈의 에너지 저장 설계"로 계층적 ESS 구조 상세화

### 제목 후보
- "왜 배터리보다 용융금속인가" (`/power/` 또는 `/why/`)
- "다이슨 모듈은 배터리가 아니라 용광로에 에너지를 저장한다"

## 8. 미해결 과제

| 과제 | 상세 |
|------|------|
| 전자기 부유 전력 소비 | 무중력 비접촉 유지에 필요한 전력 산정 (추정: 수~수십 kW) |
| 열교환기 접근 메커니즘 | 비접촉 용융체에 어떻게 열을 넣고 빼는가? 복사 열교환? 접촉식 프로브? |
| 축열조 크기 상한 | 단일 구형 축열체의 최대 크기 (전자기장 침투 깊이, 표면장력 한계) |
| 복수 소형 유닛 vs 단일 대형 유닛 | 리스크 분산 vs 복잡도 트레이드오프 |
| 모듈 회전과의 간섭 | 모듈 자체 회전(자세 제어, 인공 중력)이 부유 축열에 미치는 영향 |
| 방전 시 온도 프로파일 | ΔT가 줄어들면 터빈 효율 저하 — 최소 유효 온도 설정 |
| 축열조 위치 최적화 | 모듈 내 질량 분포·균형추 역할 겸용 가능성 |
| 용융금속 표면 방사율 관리 | MLI 반사벽의 장기 열화, 재코팅 주기 |
