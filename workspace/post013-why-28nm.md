# post013 — 왜 28nm인가

> 트리거: 자기복제 루프의 마지막 외부 의존은 AI 칩이다. 지구에서 계속 칩을 사올 거면 진정한 자기복제가 아니다. 어떤 공정으로 만들 것인가? 최첨단 3nm? 아니다 — 28nm.
>
> 선행 글: post005(왜 L5인가), post007(왜 터빈인가), post011(왜 철-니켈인가)
> 섹션: `/why/` 또는 `/computing/`
> 영문 제목 시안: "Why 28nm, Not Cutting-Edge"

---

## 핵심 논지

우주에서 AI 칩을 자급하려면 **EUV 없이 만들 수 있는 가장 강력한 공정**이 답이다. 그게 28nm. 전력 4.6배 열위는 370 MW 모듈에서 반올림 오차. 슬래그에서 실리콘이 나오고, 우주 자체가 클린룸이다.

---

## 서사 흐름

```
왜 자기복제인데 칩은 지구에서 사오나?
  → 칩도 만들어야 진짜 자기복제
    → 최첨단은 ASML 독점 + 수출규제 → 우주에서 불가능
      → 28nm는 ArF 이머전만으로 가능 (EUV 불필요)
        → Google TPU v1이 28nm로 실측 92 TOPS
          → 43장 쌓으면 H100급 — 전력 4.6배? 370 MW에서 무의미
            → 원료? 슬래그(SiO₂) → 실리콘 잉곳
              → 우주 진공 = 클린룸, 미소중력 = 대구경 잉곳
                → 팹의 95%는 우주에서, 5%는 스타십이 연 1회 배달
                  → 자기복제 루프의 마지막 고리가 닫힌다
```

---

## 본문 구성

### 1. "자기복제인데 칩은 어디서?"

- 이전 글들에서 거울, 구조재, 배터리, 열관리까지 자기복제 루프 구축
- 하지만 **AI 자율운용 칩**은 아직 지구 의존
- 칩을 못 만들면 자기복제가 아니라 보급선 의존 — post011(철-니켈)과 같은 논리
- "소행성에 리튬이 없다"가 게임 오버였듯, "우주에서 EUV를 못 만든다"가 3nm의 게임 오버

### 2. 왜 최첨단이 불가능한가

| 항목 | 28nm | 3~5nm (최첨단) |
|------|------|---------------|
| 리소그래피 | **ArF 이머전** (니콘, 캐논, ASML) | **EUV** (ASML 독점, 대당 수천억 원) |
| 장비 가용성 | 성숙 시장, 중고 풍부 | 극히 제한, 수출규제 대상 |
| 설계 복잡도 | 싱글 패터닝 | 멀티 패터닝 (극히 복잡) |
| 팹 건설 비용 | ~$3~5B | ~$20~30B |
| 수율 | 높음 (검증) | 초기 저조 |

**EUV 스캐너는 지구에서도 ASML 한 회사만 만든다.** 이걸 우주에서 만들겠다? 불가능. EUV가 필요 없는 가장 강력한 공정이 28nm.

- "7nm는?" → ArF 멀티 패터닝으로 가능하지만 설계 복잡도 폭증, 수율 급감
- "65nm면 더 쉽지 않나?" → 가능하지만 칩당 성능이 너무 낮아 물량 비용 급증
- **28nm = EUV 없이 도달 가능한 최적 밀도**

### 3. 레퍼런스: Google TPU v1 — 이건 이론이 아니다

Google이 2015년에 **실제로** 28nm로 만들어서 10만 개 이상 배포한 AI 가속기.

| 항목 | Google TPU v1 (실측) |
|------|---------------------|
| 공정 | 28nm |
| 구조 | 256 × 256 시스톨릭 어레이 |
| 연산 | 92 TOPS (INT8) ≈ 23 TFLOPS (FP16) |
| 소비전력 | 실운용 ~75W |
| 실리콘 활용률 | **90%+** |

왜 시스톨릭 어레이인가:
- GPU는 범용성 위해 제어 로직·캐시·스케줄러에 실리콘 70% 소비
- 시스톨릭 어레이는 90%+가 실제 행렬 연산 수행
- **AI만 돌릴 거면 GPU의 범용 오버헤드는 전부 낭비**

AlphaGo를 돌린 칩이다. 이론적 설계가 아니라 구글 데이터센터에서 수년간 가동된 실증 하드웨어.

### 4. H100급 달성: 물량으로 밀면 된다

NVIDIA H100: 4nm, 990 TFLOPS, 700W.

```
990 TFLOPS ÷ 23 TFLOPS = 43장

43장 × 75W = 3,225W ≈ 3.2 kW
```

| | DABEL (TPU v1 × 43장) | H100 × 1장 |
|---|---|---|
| FP16 합산 | ~990 TFLOPS | ~990 TFLOPS |
| 총 전력 | **3.2 kW** | 700W |
| 전력 비율 | **4.6배** | 기준 |

**"4.6배나 전기를 더 먹는다고?"**

다이슨 모듈 1기 = 370 MW. 3.2 kW = 370 MW의 0.00086%.
거울 면적 2.4 m²만 더 펴면 되는 차이.

지구에서는 전력이 돈이다. **우주에서 전력은 거울 면적이다.** 거울은 소행성 Fe-Ni로 만든다. 공짜.

→ 이전 글(왜 터빈인가)에서 "지구 기준 vs 우주 기준"으로 결론을 뒤집은 것과 같은 패턴

### 5. 모듈 1기 = H100 3만 장급 데이터센터

```
모듈 전력 370 MW 중 30% = 111 MW를 AI 연산에 배정

111,000 kW ÷ 0.075 kW/칩 = ~1,480,000장 (148만 장)
148만 장 ÷ 43장/H100 = ~34,000 H100 환산
오버헤드 20~30% → 보수적으로 H100 ~2.5만~3만 장급
```

현재 지구 최대급 AI 클러스터와 동급. **모듈 1기가.**
20만 기 자기복제 시 H100 수십억 장급 도달.

### 6. 원료: 슬래그에서 반도체가 나온다

```
소행성 원광 → 진공 제련
  ├→ Fe-Ni (90%+) → 구조재, 거울, 배터리
  └→ 슬래그 (SiO₂ 주성분)
       ├→ 대부분 → 방사선 차폐재 (기존 용도)
       └→ 일부 → 탄소 환원 (SiO₂ + 2C → Si + 2CO)
            → 금속 실리콘
            → 존 리파이닝 (태양열 + 진공 + 미소중력)
            → 반도체급 단결정 잉곳 (9N+)
            → 300mm 웨이퍼
```

**제련 폐기물에서 AI 칩이 나온다.** 별도 실리콘 광산 불필요.

존 리파이닝이 우주에서 유리한 이유:
- 에너지: 태양열 직접 가열 (비용 제로)
- 진공: 우주가 진공 (불순물 자동 증발)
- 미소중력: 용융대가 흘러내리지 않음 → 지구 FZ법 한계 200mm → **우주 300mm+**
- 반복 패스: 거울 각도만 조절 → 무한 반복 비용 제로

### 7. 팹: 우주 자체가 클린룸

지구 28nm 팹의 최대 비용 중 하나: 클래스 1~10 클린룸 (공기 중 파티클 극도 제거).
우주: **공기가 없다.** 파티클 오염원 자체 부재.

7대 공정 단계별 우주 적합성:

| 공정 | 우주 적합성 | 이유 |
|------|-----------|------|
| ① 잉곳 성장 | ⭐ 우주 우위 | 미소중력 FZ법 |
| ② 웨이퍼 슬라이싱 | ✅ 가능 | 기계적 공정 |
| ③ 산화/증착 | ⭐ 진공 유리 | CVD/PVD 진공 펌프 불필요 |
| ④ 포토리소그래피 | ⚠️ 병목 | ArF 스캐너·레지스트 지구 의존 |
| ⑤ 에칭 | ⭐ 진공 유리 | 챔버 단순화 |
| ⑥ 이온 주입 | ⭐ 진공 유리 | 빔 산란 감소 |
| ⑦ 배선/패키징 | ✅ 가능 | Cu 우주 조달 |

**7단계 중 6단계가 우주에서 유리하거나 동등.** 유일한 병목은 포토리소그래피 — ArF 스캐너를 한 번 올리면 수십 년 사용.

### 8. 팹 열 관리: 우주가 오히려 유리한 이유

"우주에서 반도체를 만든다고? 태양 보는 면은 수백 도, 등진 면은 영하 100도인데?"

맞다. **하지만 이건 팹 공정의 적이 아니라 설계 조건이다.** 그리고 그 설계 조건은 지구보다 오히려 유리하다.

#### 문제의 핵심: ±0.01°C

ArF 리소그래피 스캐너 내부의 투영 렌즈 시스템은 열팽창에 극도로 민감하다. 온도 0.01°C 변동 → 렌즈 곡률 변화 → 오버레이(overlay) 오차 발생 → 수율 급감. 28nm 공정의 오버레이 허용 오차는 수 nm 수준.

지구 팹에서 이 문제를 어떻게 해결하는가:
- 클린룸 전체를 23.00 ± 0.1°C로 항온 유지
- 스캐너 내부는 별도 냉각 회로로 ±0.01°C 제어
- **문제: 외부 교란이 끊임없이 들어온다** — 외기 온도 변동, 계절, 주야간, 날씨, 지진 진동, 도로 진동, 인접 장비 발열

#### 우주 팹 모듈의 열 설계

```
[팹 모듈 단면]

외부: 우주 진공 (완벽한 단열재 — 전도·대류 제로)
  │
  ├─ MLI 반사벽 (다층 반사 단열, 수십 겹)
  │    → 태양 복사열 차단율 99.5%+
  │    → 내부→외부 복사 손실도 차단
  │
  ├─ 구조 외벽 (Fe-Ni)
  │
  ├─ 능동 액체 순환층
  │    → 초순수(UPW) 또는 냉매를 미세 순환
  │    → 펌프 + 히터 + 방열 밸브로 능동 제어
  │    → 목표: 내벽 전체 23.00 ± 0.05°C 균일
  │
  └─ 팹 내부 (1 atm N₂ 분위기)
       → 장비 발열 → 순환 냉매가 흡수
       → 스캐너 내부: 전용 냉각 루프 ±0.01°C
       → 열 수지: 장비 발열 = 방열판 방출 (정상 상태)
```

#### 왜 지구보다 유리한가

| 항목 | 지구 팹 | 우주 팹 모듈 |
|------|--------|-------------|
| 외부 온도 교란 | 끊임없음 (날씨, 계절, 주야) | **없음** (진공 단열) |
| 외부 진동 | 도로, 지진, 인접 공장 | **없음** (우주 무진동) |
| 단열 비용 | 거대 HVAC 시스템 (팹 전력의 30~40%) | **진공이 공짜 단열재** |
| 열원 예측성 | 외부 교란 + 내부 장비 | **내부 장비만** (완전 예측 가능) |
| 방열 | 냉각탑, 칠러 (물, 전기 대량 소비) | **방열판** (진공 복사, 전기 불필요) |

핵심 역설: **우주의 극한 열환경(수백 도 vs 영하 100도)은 "팹 내부"에 도달하지 않는다.** 진공은 최고의 단열재이고, MLI가 복사를 차단하면 팹 내부는 외부와 열적으로 완전히 격리된다. 그 후에는 내부 장비 발열만 관리하면 되는데, 이건 지구보다 쉽다 — 외부 교란이 제로이므로.

지구 팹이 HVAC에 전체 전력의 30~40%를 쓰는 이유는 **외부와 끊임없이 싸우기 때문**이다. 우주 팹은 그 싸움 자체가 없다.

#### UPW 순환 — 바톨라이저와의 연계

팹 항온 냉매로 사용하는 초순수(UPW)는 바톨라이저 산출물:

```
바톨라이저: H₂O → H₂ + O₂ (전기분해)
역반응: H₂ + O₂ → H₂O (연료전지)

부산물 H₂O → 정제 → UPW
  ├→ 팹 항온 순환 냉매
  ├→ 웨이퍼 세정
  └→ 이머전 리소그래피 액체
```

별도 초순수 플랜트가 아니라 **바톨라이저 부산물 정제**로 충당. 물 자체가 순환한다.

#### 인공중력 구획과의 결합

이머전 리소그래피는 웨이퍼 위에 초순수 박막을 깔아야 한다 — 중력 필요. 팹 모듈의 리소그래피 구획은 오닐 실린더형 회전 구조 내부에 배치:

```
[팹 모듈 내부 구조]

진공 구획 (0G):
  ├→ CVD/PVD 증착 (진공 필수)
  ├→ 이온 주입 (진공 필수)
  └→ 플라즈마 에칭 (진공 필수)

인공중력 구획 (~1G 회전):
  ├→ ArF 이머전 리소그래피 (액체 관리에 중력 필요)
  ├→ 습식 세정 (UPW 세정에 중력 필요)
  └→ 웨이퍼 핸들링 (로봇 이송, 중력 환경이 검증됨)

웨이퍼는 진공↔인공중력 구획을 에어록을 통해 오간다.
```

회전 구획의 정밀 균형은 반도체 장비의 초정밀 진동 요건에도 유리 — 외부 진동원이 없으므로 회전 자체의 균일성만 관리하면 된다.

### 9. 외부 의존: 5%

| 구분 | 조달 | 비고 |
|------|------|------|
| 실리콘 | 현지 (슬래그) | 95% |
| 에너지 | 현지 (태양열) | 100% |
| 클린룸 | 현지 (진공) | 100% |
| 초순수 | 현지 (바톨라이저 H₂+O₂→H₂O) | 100% |
| 산소 | 현지 (바톨라이저) | 100% |
| 구리 배선 | 현지 (소행성/달) | 100% |
| ArF 스캐너 | **지구 1회** | 수십 년 수명 |
| 포토레지스트 | **지구 연 1회** | 연 수백 kg |
| 에칭 가스 | **지구 연 1회** | 재활용, 소량 |
| 도핑 원소 (B, As) | **지구 연 1회** | 수십 kg |

**스타십 1회 발사에 수십 년치 소모품 적재 가능.** 연간 보급 수 톤 이하.

"28nm 팹의 95%는 우주에서, 5%는 스타십이 연 1회 배달."

### 9. 자기복제 루프가 닫힌다

```
이전까지:
  소행성 원광 → 제련 → Fe-Ni → 거울·구조재·배터리 → 자기복제
                                                  ↑
                                          AI 칩은 지구에서 수입 ❌

이제:
  소행성 원광 → 제련 → Fe-Ni → 거울·구조재·배터리
                    → 슬래그 → Si 잉곳 → 28nm TPU → AI 자율제어
                                                  ↓
                                          자기복제 루프 완전 폐쇄 ✅
```

거울이 거울을 만들고, 배터리가 추진제를 만들고, 슬래그가 AI 칩을 만든다. **버릴 것이 없다.**

### 10. 한 줄 요약

최첨단 3nm는 ASML 독점 EUV 없이 불가능 — 우주에서 만들 수 없다. 28nm는 ArF만으로 가능하고, Google TPU v1이 실증했으며, 전력 4.6배 열위는 370 MW 모듈에서 의미가 없다. 슬래그에서 실리콘이 나오고, 우주 자체가 클린룸이다. 자기복제 루프의 마지막 고리.
