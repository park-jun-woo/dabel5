# post015 — 왜 또 SEP 추진체는 철, 니켈인가

> 트리거: 구조재도 Fe-Ni, 거울도 Fe-Ni, 배터리도 Fe-Ni — 이제 추진제까지? 너무 편의적인 거 아닌가? 아니다. Fe-Ni가 계속 나오는 건 우연이 아니라, M형 소행성에 올인한 설계의 필연이다. 그리고 추진제로서 Fe-Ni는 물리적으로도 최적이다.
>
> 선행 글: post004(채광·수송), post011(왜 철-니켈 배터리인가), post013(왜 28nm)
> 섹션: `/why/`
> 영문 제목 시안: "Your asteroid freighter's propellant is also its cargo — here's the physics"

---

## 핵심 논지

전기 추진의 전통적 추진제(Xe, Ar)는 우주에서 조달 불가. 수소는 C-type 채굴 + 전기분해 + 극저온 저장이라는 별도 인프라가 필요. Fe-Ni는 (1) 화물 자체이므로 공급 체인 제로, (2) 이온화 에너지가 Xe보다 낮고, (3) 원자량이 56~59로 수렴하여 잡탕이어도 균일 배기 가능, (4) W-TAL 하이브리드로 스러스터 하드웨어까지 현지 자급. Isp 1,800s는 Xe의 3,000s보다 낮지만, 추진제가 공짜라면 Isp 경쟁은 의미가 없다.

---

## 서사 흐름

```
"또 Fe-Ni? 너무 편의적인 거 아냐?"
  → M형 소행성에 올인하면 뭐든 Fe-Ni로 만들어야 한다
    → 추진제도 예외가 아니다
      → 근데 물리적으로 되긴 하나?
        → Fe 이온화 에너지 7.87 eV < Xe 12.1 eV — 오히려 더 쉽다
          → 원자량 56~59 수렴 — 잡탕이어도 균일 배기
            → 화물에서 16% 깎으면 추진제 조달 끝
              → "Xe는 Isp가 더 높잖아?" → 맞다, 근데 Xe는 어디서 구하나?
                → 지구에서 사온다 → 자기복제 아님
                  → Isp 경쟁 vs 공급 체인 — 우주에서의 최적화 축이 다르다
                    → W-TAL: 스러스터조차 현지 조달
                      → 추진 시스템 지구 의존 = 제로
```

## 본문 구성

### 1. "또 철-니켈?"

- 시리즈 복기: 구조재(post 3-4), 거울(post 5), 배터리(post 11) — 전부 Fe-Ni
- 독자의 합리적 의심: "뭐든 Fe-Ni라니, 결론에 맞추는 거 아냐?"
- 정직한 답: **맞다, 결론에 맞추고 있다.** M형 소행성은 90%가 Fe-Ni. 이 소행성에서 자기복제하려면 **모든 것이 Fe-Ni로 만들어지거나 포기**해야 한다
- 이건 편의가 아니라 **제약 조건이 설계를 지배하는 것** — post 11(리튬이 없으니 Fe-Ni 전지)과 동일한 패턴
- "추진제도 Fe-Ni야"는 이 논리의 극단적 귀결

### 2. 전통 추진제가 우주에서 안 되는 이유

| 추진제 | Isp | 조달 | 문제 |
|--------|-----|------|------|
| **Xe** | ~3,000 s | ❌ 지구만 | 지구 의존 = 자기복제 불가 |
| **Ar** | ~2,500 s | ❌ 지구만 (사실상) | 동일 |
| **H₂** | ~3,000-5,000 s (VASIMR) | △ C-type | 전기분해 + 극저온 저장 + 별도 소행성 채굴 |
| **Fe-Ni** | ~1,800 s | ✅ 화물 자체 | **공급 체인 제로** |

- Xe/Ar: 우주에 거의 없다. 매번 지구에서 발사해야 한다. 스타십 한 대 가격 > 추진제 가치
- H₂: C-type 소행성에 물이 있지만, 물 전기분해 → H₂ 분리 → 극저온(−253°C) 저장 → 별도 인프라 전부 필요. "추진제 공급 체인 제로"가 아니라 추진제 공급 체인을 새로 짓는 것
- **Isp가 높아봤자 조달이 안 되면 의미 없다**

→ post 7(왜 터빈인가)과 같은 패턴: 지구 기준의 최적화 축(Isp)이 우주에서는 무의미. 우주의 최적화 축은 **공급 체인 독립성**

### 3. Fe-Ni가 물리적으로 작동하는 이유

여기서 "되긴 하나?"에 대한 정량적 답변.

#### 3.1. 이온화 — Xe보다 쉽다

| 원소 | 원자량 (amu) | 1차 이온화 에너지 (eV) | 현지 조달 |
|------|------------|---------------------|----------|
| Fe | 56 | **7.87** | ✅ 화물 |
| Ni | 58 | **7.64** | ✅ 화물 |
| Co | 59 | **7.88** | ✅ 화물 |
| Xe | 131 | 12.13 | ❌ 지구 |

**Fe-Ni는 Xe보다 이온화가 40% 더 쉽다.** 홀 방전 전압(200~500V)에서 Fe, Ni, Co가 전부 이온화. 성분별 효율 차이 무시 가능.

- "이온화 에너지가 낮으면 뭐가 좋은데?" → 같은 방전 전력에서 더 많은 이온 생산 = 추력 효율 향상
- "이온화가 쉬우면 원치 않는 이온화도 쉽지 않나?" → 맞지만 홀 스러스터의 전자기장 가둠이 이를 통제

#### 3.2. 잡탕이 작동하는 이유 — 물리적 수렴

M-type 원광(Fe 85%, Ni 10%, Co 0.5%, 기타 4.5%)을 정련 없이 그대로 증발:

```
가중 평균 원자량:
  m_avg = 0.85 × 56 + 0.10 × 58 + 0.005 × 59 + 0.045 × 28
        = 55.0 amu

→ 순수 Fe(56)와 거의 동일. 잡탕이어도 배기 속도가 변하지 않는다.
```

**4가지 수렴:**
1. 이온화 에너지: 7.6~7.9 eV → 같은 전압에서 전부 이온화
2. 원자량: 56~59 amu → 배기 속도 균일
3. 끓는점: 2,862~2,927°C → ~3,000°C에서 동시 증발
4. 이온 전하: 300V에서 전부 1가 → 가속 균일

"Fe, Ni, Co가 우연히 비슷한 게 아니다 — 주기율표에서 바로 옆자리(26, 27, 28번)다. 원자 구조가 비슷하니 물리적 성질이 수렴한다."

#### 3.3. Isp 계산

```
v = η × √(2qV / m)

η = 55% (보수적)
q = 1.602 × 10⁻¹⁹ C
V = 300 V
m = 56 amu = 9.30 × 10⁻²⁶ kg

이론 배기: √(2 × 1.602e-19 × 300 / 9.30e-26) = 32,100 m/s
실효 배기: 32,100 × 0.55 = 17,700 m/s
Isp = 17,700 / 9.81 = 1,800 s
```

| 전압 | 이론 (m/s) | 실효 (m/s) | Isp |
|------|-----------|-----------|-----|
| 200V | 26,200 | 15,700 | 1,600 s |
| 300V | 32,100 | 19,300 | 1,970 s |
| 500V | 41,400 | 24,800 | 2,530 s |

보수적으로 **300V, 55% 효율 = Isp 1,800 s** 채택. Xe(3,000 s)의 60% 수준.

### 4. "Isp가 낮으면 추진제를 더 먹잖아"

맞다. 정량적으로 얼마나 차이 나는지 계산.

현재 아키텍처 기준(Δv_SEP = 3,030 m/s):

| | Fe-Ni (Isp 1,800) | Xe (Isp 3,000) |
|---|---|---|
| 배기 속도 | 17,658 m/s | 29,430 m/s |
| 질량비 MR | exp(3030/17658) = 1.187 | exp(3030/29430) = 1.109 |
| 추진제 비율 | **15.8%** | 9.8% |
| 500t 중 화물 | **404 t** | 441 t |
| 500t 중 추진제 | **76 t** | 39 t |

Fe-Ni는 추진제를 약 2배 먹는다. **하지만:**

- Xe 39톤 = 지구에서 발사. 스타십 단가 ~$50M, 39톤 전용 편당 ~$50M
- Fe-Ni 76톤 = 화물에서 깎음. 추가 비용 = **$0**

"2배 더 먹지만, 공짜 2배 vs 유료 1배 — 뭐가 나은가?"

→ 이것이 post 7(왜 터빈인가)의 반복: "효율 4.6배 열위? 370 MW에서 반올림 오차"와 동일 구조. **자원이 무한할 때 효율 경쟁은 의미를 잃는다.**

### 5. 화물이 추진제고, 추진제가 화물이다

```
화물선 500톤 출발:
  ├─ Fe-Ni 화물: 404톤 (배달 대상)
  ├─ Fe-Ni 추진제: 76톤 (화물에서 분리)
  └─ 구조+모듈: 20톤

추진제 = 화물의 일부를 깎아서 뱉는 것.
화물이 줄면 추진제도 줄지만, 필요한 Δv도 줄어든다 (질량비 자기보정).
```

이 설계의 함의:
- **추진제 탱크 불필요** — 화물 자체가 "탱크"
- **추진제 공급 체인 불필요** — 채굴, 운반, 저장, 충전 인프라 전부 불필요
- **화물선에 타기 전에 추진제가 이미 있다** — 적재 = 주유
- **잔량 관리 단순** — 화물에서 막대를 하나 더 뽑으면 됨

### 6. 스러스터도 현지에서 만든다 — W-TAL

"Fe-Ni를 뱉으면 스러스터가 버틸 수 있나?"

#### 6.1. 문제: 금속 증기가 스러스터를 죽인다

기존 홀 스러스터(SPT)의 방전 채널은 질화붕소(BN) 세라믹. Fe-Ni 증기가 BN 벽에 재응축하면 도전성 박막 → 절연 파괴 → 사망.

#### 6.2. 해답: 벽이 금속이면 금속이 쌓여도 죽지 않는다

TAL(Thruster with Anode Layer) — 러시아 TsNIIMASH, D-38/D-55/D-100 비행 실증.
채널 벽이 금속. 금속 증기가 재응축해도 **금속 위에 금속 = 전기적 특성 무변화.**

#### 6.3. 왜 텅스텐인가

"그러면 채널도 Fe-Ni로?" → 안 된다. **자기 스퍼터링.**

같은 질량의 당구공이 정면 충돌 → 에너지 100% 전달. Fe⁺(56 amu)가 Fe 벽(56 amu)을 때리면 스퍼터 수율 > 1.0 → 연쇄 증폭 → 벽이 자기 자신을 녹여 먹음.

| 충돌 쌍 | 질량비 | 스퍼터 수율 (300V) |
|---------|--------|------------------|
| Fe⁺ → Fe (자기 스퍼터링) | 1.00 | **~1.0~1.5** (폭주!) |
| Fe⁺ → W (텅스텐 벽) | 0.30 | **~0.1~0.15** |

텅스텐(184 amu)은 Fe⁺(56 amu)보다 3.3배 무겁다. 가벼운 공이 무거운 벽에 부딪히면 튕겨나간다. 스퍼터 수율 1/10.

+ 자기 차폐(magnetic shielding) → 이온의 벽 도달 자체를 100배 차단 → **수명 10만 시간+**

#### 6.4. 텅스텐은 어디서?

M형 소행성은 분화된 모체의 코어 파편. 텅스텐은 친철 원소(siderophile)로 코어에 농축.

```
Fe-Ni 진공 제련 → 존 리파이닝 → 고융점 잔류 분획 → W 농축
  → 10만 톤 배치당 W 100 kg 회수
  → 스러스터 1기 = W ~2.5 kg
  → 10만 톤에서 화물선 20대분 스러스터 제작 가능
```

**제련 부산물에서 스러스터가 나온다.** 추진제(Fe-Ni)든 스러스터(W-TAL)든 별도 공급 체인 불필요.

### 7. 종합: 추진 시스템의 지구 의존 = 제로

| 구성 | 소재 | 조달 |
|------|------|------|
| 추진제 | Fe-Ni | ✅ 화물 자체 |
| 스러스터 채널 | W | ✅ 제련 부산물 |
| 음극 | W | ✅ 제련 부산물 |
| 전력 | 태양전지 | ✅ SEL5 제작 |
| 구조 | Fe-Ni | ✅ 화물 자체 |
| 지구 의존 | | **없음** |

→ post 11(배터리도 Fe-Ni), post 13(칩도 슬래그)에 이어 **추진도 현지 자급**
→ 자기복제 루프에서 "물류"마저 루프 안에 들어온다

### 8. 한 줄 요약

Fe-Ni의 이온화 에너지(7.9 eV)는 크세논(12.1 eV)보다 낮고, 원자량(56~59 amu)은 수렴하며, 화물 자체가 추진제다. Isp가 60% 낮지만 추진제가 공짜라면 Isp 경쟁은 의미가 없다. 스러스터조차 제련 부산물(텅스텐)로 만든다. 추진 시스템의 지구 의존 = 제로.
